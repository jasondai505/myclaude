# 全球市场数据查询 · Skill 使用指南

## 这个 Skill 能做什么？

**全球市场数据查询**基于「Alpha派」全球数据库，面向海外和全球资产的结构化取数。它适合回答“指定标的 + 指标 / 数据类型 + 时间范围”的问题，常用于查询行情、历史价格、财务报表、估值指标、分析师预期、公司事件、ETF 持仓、指数成分、外汇、商品、加密资产和部分披露摘要。

本 Skill 更适合做标准化数据查询；如果需要 SEC 原文、港交所公告原文、附件、逐条 XBRL 字段或深度披露核验，应切换到专门披露 / 公告 Skill。

## 数据覆盖层级

### 美股

覆盖美国上市公司和美股市场相关的行情、基本面、事件、披露摘要和另类数据。


| 数据类型  | 可查询内容                                           |
| ----- | ----------------------------------------------- |
| 行情与价格 | 最新报价、盘前 / 盘后 / 夜盘、日线、分钟线、小时线                    |
| 公司信息  | 公司资料、同业公司、市值、股本、员工数、高管与薪酬                       |
| 财务与估值 | 三大报表、TTM 报表、财务比率、关键指标、增长率、收入分部、企业价值、DCF         |
| 分析师   | 一致预期、目标价、评级、评级上调 / 下调                           |
| 公司事件  | 财报日历、业绩 surprise、分红、拆股、IPO、并购、退市                |
| 披露摘要  | SEC filing 元数据、最新 8-K、13F 摘要、内幕交易、国会交易、ESG、融资披露 |
| 业绩纪要  | 业绩会纪要全文、最新纪要、可用公司与日期列表                          |


常见问题示例：`AAPL 最近 3 个财年的利润表`、`TSLA 最新分析师目标价摘要`、`NVDA 盘后价格`、`苹果最近一份 10-K 的提交日期和链接`。

### 全球股票

覆盖港股、日本、韩国、中国台湾、英国、加拿大、印度、德国、澳大利亚等主要海外市场股票。


| 数据类型    | 可查询内容                                  |
| ------- | -------------------------------------- |
| 行情与价格   | 最新报价、日线、分钟线、小时线                        |
| 公司与证券识别 | 股票代码、公司名、交易所、币种、CIK / ISIN / CUSIP 等标识 |
| 基本面     | 公司资料、市值、股本、三大报表、财务比率、关键指标、收入分部         |
| 市场筛选    | 按交易所、国家、行业、板块、市值、价格、成交量等条件筛选股票         |
| 公司事件    | 分红、拆股、财报日期、退市和代码变更线索                   |


全球股票最好使用本地市场后缀，例如 `0700.HK`、`7203.T`、`005930.KS`、`2330.TW`、`BHP.AX`、`VOD.L`、`SHOP.TO`、`INFY.NS`。

### ETF 与共同基金

覆盖美股 ETF、常见全球 ETF 和美国共同基金。


| 数据类型  | 可查询内容                              |
| ----- | ---------------------------------- |
| 行情与价格 | 最新报价、日线、分钟线、小时线                    |
| 产品资料  | issuer、expense ratio、AUM、基金类型、披露日期 |
| 持仓    | 持仓名称、代码、权重、前十大持仓                   |
| 配置暴露  | 国家权重、行业权重、资产暴露                     |
| 文件线索  | ETF / 基金披露搜索、按名称搜索披露               |


常见问题示例：`SOXX ETF 前 10 大持仓`、`VWO 国家配置`、`QQQ 最近一年日线价格`。

### 指数

覆盖美股三大指数、标普 500、纳斯达克、道琼斯、富时、日经、恒生等主要全球指数，以及部分区域和主题指数。


| 数据类型  | 可查询内容                           |
| ----- | ------------------------------- |
| 行情与价格 | 最新点位、涨跌幅、日线、分钟线、小时线             |
| 指数成分  | 标普 500、纳斯达克、道琼斯成分股及历史成分         |
| 市场表现  | 涨跌榜、成交活跃股票、板块 / 行业表现、板块 / 行业 PE |


常见问题示例：`标普 500 成分股`、`恒生指数过去 1 年日线`、`今天美股成交最活跃股票`。

### 外汇与贵金属

覆盖主要货币对、交叉盘、部分离岸汇率，以及黄金、白银等贵金属兑美元报价。


| 数据类型 | 可查询内容                                    |
| ---- | ---------------------------------------- |
| 行情   | EURUSD、USDJPY、USDCNY、XAUUSD、XAGUSD 等最新报价 |
| 历史价格 | 日线、分钟线、小时线                               |
| 相关新闻 | 外汇新闻、按关键词或交易对搜索新闻                        |


常见问题示例：`USDJPY 最近 30 天日线`、`XAUUSD 最新价格和涨跌幅`。

### 商品

覆盖黄金、白银、铜、原油、天然气、农产品等常见大宗商品及部分连续报价。


| 数据类型 | 可查询内容               |
| ---- | ------------------- |
| 行情   | 最新价格、涨跌幅、成交量等快照字段   |
| 历史价格 | 日线、分钟线、小时线          |
| 持仓报告 | COT 报告列表、报告数据、按日期分析 |


连续合约、滚动口径和商品代码以数据返回为准。常见问题示例：`WTI 原油过去 1 个月每日收盘价`、`铜价最近 3 个月走势`。

### 加密资产

覆盖 BTC、ETH、SOL 等主流加密资产及部分中小币种，通常使用美元计价交易对。


| 数据类型 | 可查询内容                   |
| ---- | ----------------------- |
| 行情   | 最新价格、涨跌幅、成交量等快照字段       |
| 历史价格 | 日线、分钟线、小时线，适配 7x24 小时市场 |
| 相关新闻 | 加密货币新闻、按币种或关键词搜索新闻      |


常见问题示例：`BTCUSD 最新价格`、`ETHUSD 最近 30 天日线`。

### 宏观、利率与市场基础设施

这类数据通常作为跨资产分析的辅助信息。


| 数据类型    | 可查询内容                     |
| ------- | ------------------------- |
| 经济数据    | 经济数据发布日历、海外经济指标           |
| 利率与风险溢价 | 美国国债收益率、市场风险溢价            |
| 交易时间    | 全球交易所开闭市、单交易所时间、交易所假期     |
| 资产目录    | ETF、基金、外汇货币对、商品、加密货币、指数列表 |
| 参考列表    | 交易所、国家、行业、板块、CIK 列表       |


中国宏观数据不是本 Skill 的主路径，应优先使用专门宏观 Skill。

## 适用场景

- **查行情快照**：股票、ETF、基金、指数、外汇、商品、加密资产的最新价、涨跌幅、成交量和更新时间。
- **查历史价格**：日线、分钟线、小时线，用于走势、收益率、回撤和事件前后表现。
- **查基本面和估值**：公司资料、财务报表、TTM 指标、财务比率、收入分部、DCF。
- **查 ETF / 基金暴露**：持仓、国家配置、行业配置、资产暴露和披露线索。
- **查分析师与事件**：一致预期、目标价、评级、财报日历、分红、拆股、IPO、业绩纪要。
- **查代码和标识**：公司名、ticker、ISIN、CUSIP、CIK、ADR / 本地上市变体。
- **查披露摘要和另类数据**：SEC filing 元数据、13F、内幕交易、国会交易、ESG、COT、融资披露。

## 使用须知

- **一次查询聚焦一个取数意图**：行情、财报、评级、披露摘要最好分开问。
- **写清资产类别和代码**：例如 `AAPL 美股`、`0700.HK 港股`、`QQQ ETF`、`BTCUSD`、`XAUUSD`。
- **写清时间和口径**：价格图表写日期区间和 interval；财报写年度 / 季度 / TTM；技术指标写周期长度和 interval。
- **海外股票优先使用本地后缀**：港股、日本、韩国、台湾、澳洲、伦敦、多伦多、印度等市场尤其需要。
- **披露原文不是主路径**：本 Skill 返回的是结构化摘要和线索；原文、附件、XBRL 明细应使用专门披露工具。
- **结果为空时补充条件**：优先补市场、交易所、资产类别、时间范围或指标口径；仍不命中时切换到更合适的数据路径。
