# Alpha派公开研究资料库

检索Alpha派汇聚的公开金融研究资料，涵盖A股/港股公司公告、机构研报、路演调研纪要、公众号文章、点评、基金经理观点及会议日历等多类数据源。

## 适用场景

- 查找某只股票近期发布的公告（年报、半年报、重组方案、增发公告等）
- 获取某机构/分析师对某行业或个股的研报，了解最新评级与观点
- 查阅某公司的路演纪要或调研纪要，了解管理层近期表态
- 搜索公众号精选文章，追踪特定主题或个股的市场讨论
- 查询特定机构或个股的市场点评
- 获取公募/私募基金经理在季报或年报中的最新观点
- 查看近期路演、调研、策略会、行业论坛的日程安排

## 数据覆盖范围

### 公开会议纪要

路演纪要、调研纪要（含上市公司 IR 活动）。提供 AI总结版（结构化程度高）与人工转写版两种形式。支持按个股、主办机构、时间范围过滤，覆盖 A股、港股及部分美股纪要。

### 机构研报

分三类：内资研报（A股/港股券商）、外资研报（境外机构）、第三方独立研究报告。支持按机构名、分析师、个股代码、行业、时间范围、研报类型过滤。支持图片（含图表语义描述）和表格的精细解析。

### 公司公告

覆盖 A股、港股全类型公告（年报、半年报、季报、重组、增发、分红、业绩预告等）。支持全文检索、段落级检索和表格数据提取。暂不含美股公告。

### 点评

市场点评、快评，文字点评与图片点评可区分。支持按个股代码、机构、时间范围过滤。

### 社媒（公众号）

收录 Alpha派精选的金融类公众号文章，支持按"精选"过滤。另有公众号号源元数据（号名、简介、类型等）可独立查询。

### 基金经理观点

公募/私募基金经理季报/年报观点摘录、媒体调研文章。支持按基金名称、基金经理、时间范围、行业关键词检索。注意：暂不支持直接按持仓股票代码过滤，可通过关键词全文检索。

### 会议/调研日历

四类日历：路演日历、调研日历、策略会日历、行业论坛日历。支持按时间范围、主办机构、涉及个股/行业查询。

## 说明

- **不包含私域数据**：个人录音转写、微信同步、个人上传等私有内容请使用 Alpha派私域数据库技能
- **不提供实时数据**：不支持实时行情、财务指标等结构化数据查询
- 研报图表/表格精细解析需在查询时额外说明，默认仅返回研报主信息
