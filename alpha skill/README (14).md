# A 股与指数结构化数据查询

查询 A 股上市公司（主板、科创板、创业板、北交所）、中国境内指数和全球重要股市指数的结构化定量数据，覆盖行情、财务、资本运作、指数行情、指数成分与权重等场景。

## 适用场景

- 查询个股历史价格、涨跌幅、换手率、成交量、成交额
- 调取财务三表及单季度拆分数据
- 查询盈利、偿债、成长、营运能力等财务指标
- 查询 PE、PB、PS、PCF、EV/EBITDA、股息率等估值指标
- 查看股本结构、十大股东、十大流通股东、质押冻结情况
- 查询分红送转、回购、增减持、定增配股、业绩预告（预增/预减/扭亏等）等公司行为
- 查询融资融券、沪深港通资金流向、概念板块表现
- 查询指数基本信息、指数行情、指数成分券、指数权重、指数财务指标
- 查询申万、中信、长江证券等行业指数行情和估值表现
- 查询港股、美股、日韩台、欧洲等全球重要股市指数行情

## 数据覆盖范围

### 行情与市场数据

- **个股日线行情**：开盘、最高、最低、收盘、成交量、成交额、涨跌幅、换手率
- **市场表现**：多周期涨跌幅、振幅、Beta、波动率、相对强弱
- **估值指标**：PE、PB、PS、PCF、EV/EBITDA、股息率、市值
- **融资融券**：融资余额、融券余额、融资买入额、融券卖出量
- **沪深港通资金**：北向（沪股通+深股通）、南向每日净流入
- **概念板块行情**：概念板块每日涨跌表现

### 财务报表

- **资产负债表**：资产、负债、所有者权益各科目
- **利润表**：收入、成本、费用、利润各科目
- **利润表单季值**：单个季度非累计利润表数据
- **现金流量表**：经营、投资、筹资活动现金流科目
- **所有者权益变动表**：权益项目期初、变动、期末余额

### 财务指标与附注明细

- **综合财务指标**：盈利能力、偿债能力、成长能力、营运能力等等指标
- **单季度财务指标**：按单季拆分的核心财务比率
- **财务衍生指标**：ROE、ROA、毛利率、净利率、资产负债率、流动比率等
- **财务附注明细**：应收账款、预付款、存货、固定资产、无形资产、长期借款、非经常性损益、研发投入、税费、关联交易等

### 公司基本信息

- 公司基本资料、上市信息、子公司信息
- 董监高任职信息、薪酬与持股
- 雇员概况、学历分布
- 首发上市计划与结果
- 定期报告预约披露时间

### 股本、股东与公司行为

- **股本结构**：总股本、流通股、限售股及历史变化
- **股东信息**：十大股东、十大流通股东、参控股公司、证券投资持仓
- **分红送转**：历次分红方案、年度分红统计、股利支付率
- **公司行为**：增减持、回购、质押冻结、增发配股、重大资产交易、对外投资、关联交易、诉讼仲裁、违规处罚、担保抵押、要约收购、停复牌
- **业绩预告**：预增、预减、扭亏、预盈、预亏、减亏、持平、预警等类型
- **员工持股**：员工持股计划、资金规模、股票来源、参与对象

### 科创板专有数据

科创板公司拥有独立数据体系，覆盖行情、估值、财务三表、财务指标、分红、股本股东、基本信息、经营分析、业绩预告等。查询科创板公司时应明确科创板身份。

### 指数数据

- **指数基本信息**：指数名称、代码、简称、分类、发布机构、发布日期、基期、基点、样本空间、选样标准、加权方式、调整周期、指数简介
- **指数行情**：通过指数名称、简称、代码或分类定位指数，并查询开盘、最高、最低、收盘、涨跌幅、成交量、成交金额
- **全球重要股市指数行情**：查询港股、美股、日韩台、欧洲、澳洲、印度、东南亚等主要股指的开盘、最高、最低、收盘、涨跌和涨跌幅
- **指数成分**：指数成分券、启用日期、剔除日期、当前样本状态、样本所在市场
- **指数权重**：指数成分券在指定日期或最近可得日期的权重
- **指数财务指标**：指数每股收益、每股净资产、净资产收益率、销售毛利率、销售净利率等
- **行业指数**：申万、中信、长江证券等行业指数行情；可按指数名称或简称查，如申万一级行业、中信汽车、能源(长江)等
- **全球指数示例**：恒生指数、恒生科技、恒生国企、标普500、纳斯达克100、纳斯达克综合、道琼斯工业、日经225、韩国KOSPI、台湾加权、英国富时100、德国DAX、法国CAC40等
- **常见指数体系**：上证、深证、国证、中证、沪深港、港股通、香港、MSCI、中债、中华交易服务、AMAC、全国股转、期货、基金等指数
- **常见指数类型**：股票指数、债券指数、可转债指数、组合资产指数、复合指数、基金指数、期货指数、其他指数
- **常见主题示例**：中证全指、中证新兴、中证A500、中证1000、沪深300、上证50、科创50、创业板指、新能源车、半导体、人工智能、碳中和、红利低波、港股通、沪深港、全球半导体等
