# 事实辩论

对用户提出的投研观点进行多维度的事实检验与结构化质疑，帮助投资者识别论点漏洞、完善分析框架。

## 适用场景

- 想对自己的投资逻辑进行压力测试
- 需要 Devil's Advocate 视角反驳一个行业或公司判断
- 在投委会或路演前预演可能遭受的质疑
- 验证某个观点的数据支撑是否充分

## 输入

用户的投研观点或论点，例如：

- "XX 公司今年利润增长确定性强"
- "新能源车渗透率还有很大提升空间"
- "美联储降息对港股是重大利好"

可涉及个股、行业、宏观等任意层面。

## 输出

结构化质疑报告，包含：

- **论点还原** — 精炼归纳用户的核心主张
- **事实检验** — 逐条核查支撑数据是否准确、是否存在反例
- **逻辑漏洞** — 识别论证链条中的因果谬误或遗漏假设
- **反向证据** — 列举与该观点相悖的市场数据或历史案例
- **风险情景** — 列出使该观点失效的关键条件变化

## 说明

本技能不输出最终投资判断，目的是帮助用户把观点想得更严密，而非否定观点本身。
