# 竞对与可比公司分析

对目标公司进行深度竞争对手识别与横向对比，输出涵盖业务、经营、财务、估值的完整竞对分析报告。

## 适用场景

- 分析一家公司的核心竞争对手及各自竞争优势
- 比较同行业可比公司的财务指标与估值水平
- 在投委会或路演中展示公司的行业相对位置
- 筛选投资标的时对标行业内最优秀的公司

## 输入


| 字段   | 是否必填 | 说明                    |
| ---- | ---- | --------------------- |
| 公司名称 | 必填   | 目标公司                  |
| 股票代码 | 必填   | 如"600519.SH"          |
| 关注焦点 | 选填   | 如"重点关注出海业务"、"聚焦毛利率差异" |


## 输出

完整 Markdown 竞对分析报告，包含：

1. **竞对公司识别** — 智能筛选最相关的可比公司，附相似度评分（五星制）
2. **公司分析叙述** — 每家公司的核心竞争力与差异化定位
3. **业务经营对比表** — 营收规模、增速、市场份额、核心业务拆分横向对比
4. **财务与估值对比表** — ROE、毛利率、净利率、PE/PB/PS 等核心指标对比
5. **竞争格局总结** — 行业竞争态势与目标公司的相对优劣势判断

## 覆盖范围

支持 A 股、港股、美股的跨市场可比公司对比分析。
