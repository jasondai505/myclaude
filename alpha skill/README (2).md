# 调研问题生成

为机构投资者准备上市公司调研问题清单，按投研框架系统化分类，标注优先级，帮助调研者把有限时间用在最有价值的问题上。

## 适用场景

- 调研前准备提问列表，避免临场遗漏关键问题
- 业绩会、路演、电话会前系统梳理待确认事项
- 对新覆盖标的建立初步的调研问题库

## 输入


| 字段   | 是否必填            |
| ---- | --------------- |
| 公司名称 | 必填              |
| 股票代码 | 必填              |
| 关注重点 | 选填（如"重点关注海外扩张"） |


## 输出

结构化调研问题报告，按 7 大维度分类：


| 维度   | 典型问题方向            |
| ---- | ----------------- |
| 战略方向 | 中长期战略、新业务布局、管理层意图 |
| 经营现状 | 订单、产能、价格、市场份额     |
| 财务质量 | 利润含金量、现金流、应收账款    |
| 行业竞争 | 竞争格局变化、护城河稳定性     |
| 风险因素 | 政策风险、客户集中度、技术替代   |
| 资本运作 | 分红回购计划、融资安排、股权结构  |
| 市场分歧 | 针对市场争议点的直接提问      |


每条问题附**星级评定**（★ 到 ★★★），标注问题的重要性与当前时效性，帮助调研者快速决定提问优先级。

## 说明

问题清单基于公司最新公告、财务数据、卖方研究和市场关注点动态生成，而非固定模板，确保问题的针对性和时效性。
