# 公募基金深度研究

面向公募基金的一站式深度分析与研究报告生成，覆盖单基金研究、基金经理画像、基金公司分析、多基金对比及组合构建等场景。

## 适用场景

- 拿到一只基金代码或名称，想快速了解其投资风格、业绩归因与风险特征
- 需要为某位基金经理建立完整画像，理解其投资框架、能力圈边界与言行一致性
- 评估某家基金公司的投研体系、经理团队稳定性与产品线布局
- 在两只或多只基金之间做选择，需要同维度对齐的业绩/持仓/风格/风险对比
- 基于现有持仓或候选基金，构建或优化基金组合，识别风格冗余与配置缺口

## 输入


| 字段      | 是否必填  | 说明                           |
| ------- | ----- | ---------------------------- |
| 基金代码/名称 | 视场景必填 | 支持单只或多只基金，如"720001"或"财通价值动量" |
| 基金经理姓名  | 视场景必填 | 如"张坤"，用于经理深度画像               |
| 基金公司名称  | 视场景必填 | 如"易方达基金"，用于公司研究              |
| 关注维度    | 选填    | 如侧重风险分析、持仓结构或业绩归因；未指定时生成完整报告 |


## 输出

根据用户意图自动匹配分析模块，输出完整研究报告：


| 分析类型    | 报告结构                         | 典型场景                  |
| ------- | ---------------------------- | --------------------- |
| 基金一页纸   | 十三章框架，涵盖概况、业绩、风险、持仓、归因、经理画像等 | "帮我分析这只基金"            |
| 基金经理画像  | 八章框架，覆盖投资框架、能力圈、言行一致性、风格演变   | "金梓才这个经理怎么样"          |
| 基金公司研究  | 七章框架，覆盖投研体系、经理团队、产品线、行业配置特征  | "华夏基金怎么样"             |
| 基金对比分析  | 八章框架，业绩/持仓/风格/风险同维度对齐        | "720001 和 110011 选哪个" |
| 组合构建与优化 | 七章框架，覆盖相关性穿透、冗余检测、配置建议       | "帮我搭一个稳健型组合"          |


报告以 Markdown 文件形式交付，支持直接阅读或进一步编辑。

## 说明

- 本技能专注于**分析框架与报告生成**，底层数据查询由配套的数据技能提供支持
- 需要用户提供准确的基金代码、名称或经理/公司信息，不支持模糊语义识别
- 不适用于单一数据查询（如仅查净值走势、实时排名或某一天的涨跌幅）
- 报告基于最新可获取的公开披露数据（季报、半年报、年报），可能存在 1-2 个月的披露滞后
