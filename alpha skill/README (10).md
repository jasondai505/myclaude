# 全球资本市场日报

生成涵盖全球主要市场的综合日报，为机构投资者提供每日开盘前或收盘后的全景市场概览。

## 适用场景

- 每日晨会前快速了解隔夜海外市场动态
- 收盘后汇总全球市场表现与核心驱动因素
- 了解宏观政策、地缘事件对各市场的影响

## 输出结构


| 板块   | 内容                   |
| ---- | -------------------- |
| 市场行情 | 主要股指、债券、商品、汇率涨跌概览    |
| 宏观动态 | 重要经济数据发布、央行政策动态      |
| 策略观点 | 市场主流机构的策略判断与仓位变化     |
| 主题追踪 | 当日热点主题（AI、新能源、地缘政治等） |
| 个股聚焦 | 重要个股异动与事件驱动分析        |
| 会议纪要 | 当日重要路演/业绩会核心观点摘要     |


## 支持模式

- **标准模式**：覆盖全球主要市场的客观日报
- **个性化模式**：基于用户关注的行业/主题/标的，优先呈现相关内容

## 覆盖市场

美股、港股、A 股、欧股、日股；黄金、原油等大宗商品；美元、人民币等主要汇率；美债、中债收益率。
