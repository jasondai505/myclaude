# 宏观事件分析

对关税政策、央行加息/降息、地缘冲突、大选、汇率剧变、大宗商品冲击等宏观事件进行六步结构化深度分析，输出机构级投研框架报告。

## 适用场景

- 重大宏观事件发生后，需要快速梳理市场影响和交易应对
- 想系统推演一个宏观冲击的传导路径（如：关税升级会如何影响A股、美债、人民币）
- 需要对比历史上的类似事件，找到可参照的资产表现规律
- 对某个宏观场景进行情景推演，评估基准/乐观/悲观三种路径的概率和资产方向
- 希望将宏观判断落地成具体的交易组合和止损规则

## 输入


| 字段      | 是否必填 | 说明                                   |
| ------- | ---- | ------------------------------------ |
| 宏观事件描述  | 必填   | 如"美联储意外加息50bp"、"中美关税升至145%"、"俄乌停火谈判" |
| 关注的资产类别 | 选填   | 如"A股、人民币、黄金"；不填则覆盖主要大类资产             |
| 地域范围    | 选填   | 如"重点看中国市场"；不填则全球视角                   |


## 输出

包含以下六大模块的完整 Markdown 分析报告，附可视化图表（传导路径图、历史案例对比图）：

1. **关键变量** — 驱动事件演化的 3–7 个核心变量，含当前读数与关键阈值
2. **传导路径** — 从事件冲击到资产定价的因果链，标注传导时滞与放大/衰减机制
3. **历史类似案例** — 2–4 个可比历史案例，含量化资产回报数据与本次异同分析
4. **情景推演** — 基准/乐观/悲观三情景及概率权重，各情景下主要资产的方向判断
5. **交易表达** — 可执行的交易组合，含标的、方向、仓位比例、入场条件与止损
6. **失效条件** — 3–5 个可量化的框架失效信号，附止损/退出规则

## 说明

本技能专注宏观事件的系统性研判与交易推演，不适用于：个股基本面分析、行业研究报告生成、单纯行情或财务指标查询。
