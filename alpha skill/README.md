# 公司一页纸

为上市公司（A股/港股/美股）生成买方视角的投研快速了解报告，帮助投资者在最短时间内建立对一家公司的全面认知。自动识别市场类型并采用对应的数据采集流程和报告模板。

## 适用场景

- 首次覆盖一家公司，需要快速构建投研框架
- 开会前快速熟悉标的基本面
- 整理已有认知，形成结构化的一页纸文档
- 了解某家美股公司与 A/H 股同类公司的对比关系

## 输入


| 字段   | 是否必填 | 说明                       |
| ---- | ---- | ------------------------ |
| 公司名称 | 必填   | 如"贵州茅台"、"NVIDIA"         |
| 股票代码 | 必填   | A股如 600519.SH，美股如 NVDA.O |
| 输出语言 | 选填   | 默认中文，美股可选 English        |


## 市场识别

根据股票代码后缀自动判断：

- `.SH` / `.SZ` / `.BJ` → A股
- `.HK` → 港股
- `.O` / `.N` → 美股

## 输出模块

### A股/港股报告（10大模块）

1. **近况跟踪** — 近期重要事件与公告摘要
2. **投资逻辑** — 核心驱动因素与催化剂
3. **竞争力分析** — 护城河、壁垒与行业地位
4. **业务拆分** — 各业务线收入结构与毛利拆解
5. **产销链** — 上下游关系与议价能力
6. **管理层** — 核心管理团队背景与持股情况
7. **财务分析** — 历史财务数据与趋势分析
8. **调研大纲** — 针对该公司的关键调研问题
9. **估值预测** — 主流估值方法与目标价参考
10. **风险提示** — 核心风险因素梳理

### 美股报告（12大模块）

1. **公司简介** — 业务概述、市场地位、市值区间
2. **近况跟踪** — 近期重要事件与财报亮点
3. **投资逻辑** — 短期/长期逻辑与核心竞争力
4. **业务拆分** — 发展历程与各业务线收入结构
5. **产销链** — 上下游关系与客户集中度
6. **财务数据** — 历史收入、利润、现金流趋势
7. **市场重点关注** — 机构投资者核心关注问题
8. **行业对比** — 与主要竞争对手的横向比较
9. **A/H 股映射** — 对应的 A 股或港股同类标的
10. **市场分歧** — 多空双方核心论点对比
11. **估值预测** — 盈利预测与一致预期
12. **Investment Takeaways** — 核心投资结论

## 数据来源

- **A股/港股**：综合调取卖方点评、年报/半年报公告、财务数据库及宏观行业信息，多源交叉验证
- **美股**：综合调取 SEC 公告、卖方研报、财务数据库、市场新闻及社交媒体观点，提供有数据支撑的全景视图
