# 历史复盘

对指定股票、板块或指数进行历史走势复盘，解析每段涨跌区间背后的驱动逻辑，并生成带注释的可视化走势图。

## 适用场景

- 复盘某只股票过去一段时期的完整走势
- 分析一段历史行情中各波段的驱动因素
- 研究历史上某类催化剂（如加息、政策出台）对特定标的的影响
- 向新同事或投委会展示标的历史投资回顾

## 支持标的类型

- **个股**：A 股、港股、美股上市公司
- **指数**：上证指数、沪深 300、恒生指数、纳斯达克等主要指数
- **板块**：申万(一级/二级)行业，板块（指数）

## 输入

- **标的名称或代码**（必填，默认上证指数）
- **时间区间**（选填，默认最近 2 年）

## 输出

1. **分段复盘表格** — 将历史行情拆解为若干波段（上涨/下跌/震荡），逐段标注时间区间、涨跌幅、核心驱动事件
2. **分段走势图** — 带波段标注的 K 线或收益率走势图，直观呈现每段行情的起止与归因
3. **信息来源标注** — 每段解释均附来源，无法归因的区间明确标记

## 说明

复盘聚焦已发生的客观事件归因，不包含对未来走势的预测或投资建议。
