# 订阅公众号信息流与摘要

浏览并整合用户在 Alpha 派订阅的微信公众号推送，支持按时间、按号、按主题或个股收窄，将多篇文章合成主题简报，帮助投资者高效消化订阅精华。

## 适用场景

- 查看今日/本周订阅公众号的新推送汇总
- 指定几个公众号，按时间倒序列出标题
- 检索与某行业或某公司相关的订阅推送
- 将同一主题的多篇文章合并为一份简报（带来源号与标题）
- 复盘某时段漏看的重点内容，挑选行业热点

## 输入

- 时间范围（今天、本周、最近 N 天、上周等；模糊时默认最近 3 天）
- 可选：指定公众号名称列表
- 可选：主题关键词或个股名称/代码

## 输出

根据用户意图，输出以下形态之一：


| 形态       | 说明                         |
| -------- | -------------------------- |
| 按公众号分组列表 | 各号标题按时间倒序，附发布时间            |
| 主题合并简报   | 多文合成，结论句 + 加粗关键数据，标注来源号与标题 |
| 漏看复盘     | 指定时段内按重要性挑选 N 条，含摘要与来源     |
| 热点日报     | 财经深度摘要 + 生活类一句话概述          |


## 说明

仅检索用户在 Alpha 派中**已订阅**的微信公众号文章，不混入公开研报库或公域搜索结果。正文片段不足时会提示并建议阅读原文，不编造内容。
