# 港股结构化数据查询

查询香港联交所上市公司各类定量数据，覆盖行情、财务、股本结构及持股信息，财务报表区分金融与非金融两套体系。

## 适用场景

- 查询港股个股的历史价格、成交量、多周期涨跌幅
- 调取港股财务报表（自动区分金融/非金融报表结构）
- 查看主要股东持股比例、董事持股变动
- 了解股息率、PE/PB 等估值数据及分红派息记录

---

## 数据覆盖范围

### 行情与市场数据

- **日线行情**：开盘/最高/最低/收盘价、成交量、成交额、成交均价、涨跌幅
- **多周期表现**：5日/20日/60日/年初至今涨跌幅、振幅、换手率、Beta、对数收益率、连续涨跌天数
- **估值指标**：PE（静态/滚动TTM）、PB、PS、PCF、PEG、股息率、总市值、流通市值

### 财务报表（金融 / 非金融两套体系）

港股财务报表按公司类型区分，查询时自动匹配：

**非金融公司（制造、科技、消费、医药等）**


| 报表    | 主要科目                                       |
| ----- | ------------------------------------------ |
| 利润表   | 营业收入、营业成本、毛利、行政开支、销售费用、经营利润、归属股东净利润、每股基本盈利 |
| 资产负债表 | 流动/非流动资产与负债、总资产、总负债、股东资金、现金及等价物            |
| 现金流量表 | 经营/投资/筹资活动现金流、期初期末现金余额                     |


**金融公司（银行、保险、券商）**


| 报表    | 主要科目                         |
| ----- | ---------------------------- |
| 利润表   | 净利息收入、非利息收入、营业支出、拨备、归属股东净利润  |
| 资产负债表 | 客户存款、贷款及垫款、投资证券、总资产、总负债、股东资金 |
| 现金流量表 | 同非金融                         |


> 港股多数公司只有中报+年报，无季报。

### 财务指标

**非金融公司衍生指标**：毛利率、净利率、ROE、ROA、资产负债率、权益乘数、流动/速动比率、营收增长率、净利润增长率、每股净资产

**金融公司衍生指标**：资本充足率、成本收入率、净息差、ROE、ROA、每股净资产

**最新摊薄指标（通用）**：每股净资产、每股收益、流动比率、应收/应付账款周转、存货周转

### 主营业务收入分布

- 按**产品/业务线**分布的营业收入明细
- 按**地区**分布的营业收入明细

### 股本与股东

- **股本结构**：法定股数、实收股数、股本变动明细
- **主要股东持股**：股东名称、直接/间接持股数、持股比例、权益性质
- **董事持股权益**：董事姓名、持股数量、权益金额

### 分红派息

每股派息金额、货币、除权除息日、过户日期、派息日期、分红说明

### 公司基本信息

公司名称（中英文）、注册地、成立日期、上市日期、上市板块、核数师、申万行业分类（一级）、所属赛道（金融/周期/制造/科技/医药/消费）

### 董监事信息

董监事会成员姓名、职务、任职日期

## 港股 vs A 股主要差异


| 维度    | 港股             | A 股          |
| ----- | -------------- | ------------ |
| 财报频率  | 中报 + 年报（多数无季报） | 季报 + 中报 + 年报 |
| 报表体系  | 金融/非金融两套       | 统一报表         |
| 货币单位  | 港币/人民币/美元不统一   | 统一人民币        |
| 行业分类  | 申万一级 + 赛道      | 申万一/二/三级     |
| 扣非净利润 | 无此概念           | 有            |
| 业绩预告  | 无强制披露          | 有            |
