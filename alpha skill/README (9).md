# 公告点评

对上市公司近期发布的公告、业绩报告或重要事件，生成专业机构视角的结构化点评报告。

## 适用场景

- 公司发布年报、季报、业绩预告后，需要快速生成点评
- 重大事件（并购重组、股权激励、重要合同）发生后的分析解读
- 需要以买方或卖方视角输出研究观点

## 输入


| 字段    | 是否必填 | 说明                         |
| ----- | ---- | -------------------------- |
| 公司名称  | 必填   | 如"贵州茅台"                    |
| 股票代码  | 必填   | 如"600519.SH"               |
| 点评类型  | 选填   | 业绩点评 / 事件点评 / 公告点评（默认自动判断） |
| 视角    | 选填   | 买方（默认）/ 卖方                 |
| 自定义模版 | 选填   | 粘贴模版文本或指定文件路径              |
| 关注焦点  | 选填   | 指定特别关注的分析方向                |


## 输出

结构化 Markdown 点评报告，包含：

- 事件概述与核心数据
- 业绩拆分与驱动因素分析
- 投资逻辑更新与风险提示
- 盈利预测与估值判断
- 投资建议

## 覆盖范围

支持 A 股主板、科创板、创业板上市公司。数据来源涵盖公告原文、卖方点评、财务数据库，确保点评内容有据可查。

支持用户自定义模版——提供模版后严格按照章节结构填充内容，不改动格式骨架。
