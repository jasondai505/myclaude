# EDB 宏观与行业数据查询

查询宏观经济及行业专题的时序指标数据，覆盖中国及全球主要经济体，数据库存量约 143 万条指标。

## 适用场景

- 查询 GDP、CPI、PPI、进出口等宏观经济指标
- 了解利率、汇率、货币供应量（M1/M2）的历史走势
- 查询特定行业的产销量、价格或进出口数据
- 对比不同省市的经济统计数据

## 数据覆盖

**宏观经济**

- 中国：GDP、CPI、PPI、PMI、进出口、财政收支、社融、利率、汇率
- 全球主要经济体：美国、欧盟、日本、印度、中国香港等

**地区经济**

- 中国分省市、区县经济统计数据

**行业专题（21 个行业）**

- 汽车、能源、钢铁、化工、医药生物、房地产等行业的产销量、价格、进出口数据

**国际机构**

- 联合国人口统计、IMF 数据、WIPO 知识产权统计

## 说明

本技能专注于数据库内的结构化时序数据查询，返回具体数字与时间序列。不适用于个股财报数据、研报纪要内容检索，或市场规模/竞争格局等需要主观分析的问题。
