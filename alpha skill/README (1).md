# 行业一页纸

为指定行业生成买方视角的投研快速了解报告，帮助投资者系统建立行业认知框架。

## 适用场景

- 首次研究一个新行业，需要快速构建分析框架
- 开会前快速熟悉行业基本面与竞争格局
- 整理行业研究成果，形成结构化文档

## 输入

- **行业名称**（必填），例如"工业母机"、"创新药"、"光伏组件"

## 输出模块

生成完整 Markdown 报告，涵盖以下核心模块：

1. **行业定义与边界** — 行业范畴、细分赛道划分
2. **产业链全景** — 上中下游结构、关键环节价值分布
3. **市场规模** — 总量、增速及未来空间测算
4. **竞争格局** — 主要玩家市场份额与竞争态势
5. **核心壁垒** — 技术壁垒、规模壁垒、资质壁垒等分析
6. **驱动因素** — 供需驱动、政策催化、技术变革
7. **主要上市公司** — 代表性标的及各自定位
8. **投资风险** — 行业层面的系统性风险梳理
9. **关键指标** — 追踪该行业景气度的核心数据指标

## 数据来源

综合调取卖方行业研报、宏观数据库、产业链调研资料，提供有数据支撑的行业视图。
