# 每日涨跌复盘

生成指定市场某交易日的涨跌榜复盘，解析当日主要上涨/下跌个股背后的驱动逻辑。

## 适用场景

- 每日收盘后快速了解当日市场涨跌结构
- 追踪关注的行业或主题今日表现
- 复盘重要涨跌股的事件驱动与资金动向

## 支持市场


| 市场  | 说明         |
| --- | ---------- |
| A 股 | 沪深两市（默认）   |
| 港股  | 香港联交所      |
| 美股  | 纽交所 / 纳斯达克 |


## 输入

- **市场**：A 股 / 港股 / 美股（默认 A 股）
- **日期**：指定交易日或默认最近一个交易日
- **个性化模式**：可基于用户关注的行业/主题，优先展示相关标的

## 输出

结构化涨跌复盘报告，包含：

- 当日涨幅榜与跌幅榜，默认各 `10` 只，总 `Top N = 20`
- 默认筛选口径为涨幅 `>= 5%` 与跌幅 `<= -5%`，两个榜单均按成交额排序
- 每只股票的涨跌原因（公告事件、行业政策、资金驱动等）
- 个性化模式下会说明是否优先展示与用户兴趣相关的标的，并在不足时按全市场同方向口径补足

每只股票默认至少经过一次 `search_paipai` 查询和一次 `search_web` 查询交叉验证；无据可查的涨跌原因会明确标记为"原因未完全确认"。
