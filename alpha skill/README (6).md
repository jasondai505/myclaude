# 主题选股

基于用户输入的投资主题，从 A 股上市公司中筛选产业链相关标的，输出按产业链位置分类的结构化选股表格。

## 适用场景

- 某个行业主题兴起，想快速梳理 A 股受益标的
- 行业会议或政策出台后，寻找相关产业链公司
- 构建主题投资组合前的候选标的初筛

## 适用主题类型

支持任意投资主题，例如：

- 科技赛道：AI 应用、商业航天、机器人、低空经济、半导体设备
- 新能源：储能、光伏、固态电池、绿氢
- 消费与医药：银发经济、医疗器械出海、创新药
- 宏观驱动：国产替代、"一带一路"受益、出口链

## 输入

直接描述主题即可，例如：

- "A 股哪些公司受益于低空经济"
- "梳理人形机器人产业链上的公司"
- "帮我找商业航天相关标的"

## 输出

纯 Markdown 选股表格，包含以下字段：


| 列名     | 说明                |
| ------ | ----------------- |
| 一级分类   | 产业链上/中/下游或核心/边缘受益 |
| 二级分类   | 具体细分环节            |
| 公司简称   | A 股上市公司名称         |
| 股票代码   | 含交易所后缀            |
| 题材纯度 ★ | 与主题相关程度（1-5 星）    |
| 入选理由   | 该公司进入名单的核心逻辑      |
| 最新进展   | 来自公告/研报/互动易的近期信息  |


数据来源涵盖公告、研报、互动易问答、卖方点评，确保筛选结果有据可查。
