# Alpha派市场情绪数据库

查询Alpha派独家生产、用于刻画A股/港股机构市场情绪的系列数据，覆盖每日机构热搜股票排行与蓝宝书（机构热议主题梳理）国内版和全球版。

## 适用场景

- 想知道今天/昨天机构在热议哪些题材和板块（蓝宝书）
- 想了解公募、私募、保险机构当前最热门的股票是哪些
- 查询某一天的蓝宝书晨报、午报或晚报内容
- 对比本周与上周蓝宝书主题变化，把握市场情绪边际移动
- 查看全球版蓝宝书，了解国际机构热议的投资主题

## 输入


| 字段      | 是否必填 | 说明                                  |
| ------- | ---- | ----------------------------------- |
| 查询类型    | 选填   | 蓝宝书 或 机构热票；不指定时根据关键词自动判断            |
| 日期或时间范围 | 选填   | 支持具体日期、今天/昨天/前天、本周/上周；默认返回最新一期      |
| 批次      | 选填   | 晨报、午报、晚报（仅蓝宝书国内版有效）；不指定时返回最新批次      |
| 版本      | 选填   | 国内版 或 国际版/全球版/美版；不指定时同时返回两个版本       |
| 机构类型    | 选填   | 公募、私募、保险（仅机构热票查询时有效）；不指定时返回全部机构汇总排行 |


## 输出

**蓝宝书**：按热度从高到低排列的题材列表，每条包含：

- 题材名称与热度（🔴 数量表示热度高低）
- 题材溯因：机构热议该主题的详细背景与逻辑
- 关联个股：该主题覆盖的相关股票（名称 + 代码）

**机构热搜股票**：股票热度排行表，包含：

- 排名、股票名称、股票代码、热度值

## 数据覆盖范围

### 蓝宝书·国内版


| 维度   | 说明                            |
| ---- | ----------------------------- |
| 更新频率 | 每日三个批次：晨报、午报、晚报               |
| 题材内容 | 题材名称、热度高低、溯因描述（机构讨论逻辑与背景）     |
| 关联个股 | 每个题材对应的 A股/港股成分股列表（名称 + 证券代码） |
| 历史查询 | 支持按日期或时间范围查询历史数据              |
| 对比分析 | 支持跨日期、跨批次的主题变化对比              |


### 蓝宝书·国际版（全球版 / 美版）


| 维度   | 说明             |
| ---- | -------------- |
| 更新频率 | 每日一个版本，无批次区分   |
| 题材内容 | 题材名称、热度高低、溯因描述 |
| 关联个股 | 暂无（国际版不含成分股数据） |


### 机构热搜股票


| 维度   | 说明                   |
| ---- | -------------------- |
| 机构类型 | 公募、私募、保险，支持单独查询或三类汇总 |
| 内容字段 | 股票名称、股票代码、热度值        |
| 排序方式 | 按热度值降序排列             |


## 说明

- 本技能专注于Alpha派独家市场情绪数据，**不适用于**获取实时行情数据、财务指标、公开新闻资讯
- 蓝宝书国际版不包含关联成分股，仅提供主题与溯因描述
- 若未指定蓝宝书版本，默认**同时**返回国内版和国际版结果
