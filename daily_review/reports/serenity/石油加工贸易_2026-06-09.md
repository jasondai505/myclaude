## 第一层：全球供应链卡脖子分析

### 产业链全貌

从最终产品往下游反推：

1.  **最终应用层**：智能手机（芯片光刻胶）、电动车（电池电解液/轮胎）、光伏板（EVA封装膜）、医药（药瓶/输液袋）、风电叶片（碳纤维原丝）、高性能飞机/火箭（碳纤维/特种树脂）。 *（补充：这是终端需求驱动力）*
2.  **化工新材料/特种化学品层**：光刻胶（树脂/感光剂）、湿电子化学品（高纯溶剂）、特种工程塑料（PEEK/PI）、碳纤维原丝（PAN基）、高端聚烯烃（POE/茂金属PE）、高吸水性树脂（SAP）。 *（补充：价值最高、技术壁垒最高的层级）*
3.  **基础化工原料层**：乙烯、丙烯、纯苯、PX（对二甲苯）、乙二醇、丁二烯。 *（补充：这是连接炼油与化工的“中间体”）*
4.  **炼油与裂解层**：通过常减压、催化裂化、加氢裂化、蒸汽裂解等工艺，将原油转化为石脑油、汽油、柴油、煤油以及上述基础原料。 *（BOM系统已有）*
5.  **原油贸易与物流层**：VLCC（超级油轮）运输、全球油库仓储、管道输送（如Druzhba管道）、期货/期权套保。 *（补充：这是个重资产、高杠杆、对地缘高度敏感的层级）*
6.  **原油生产/勘探层**：中东（沙特阿美）、北美（页岩油）、俄罗斯（Rosneft）、深海（巴西国家石油/圭亚那）。 *（BOM系统已有）*

### 供需缺口量化

-   **需求端**：全球乙烯需求每年稳定增长3-4%（主要来自包装和精细化学品）。而高端材料，如光刻胶用树脂、导电胶粘剂、碳纤维原丝，需求增速在 **10-20%**，且附加值极高。
-   **供给端**：全球炼油产能过剩（尤其是中国沿海的大型炼化基地）。但在 **特种催化剂** 和 **高性能化工新材料** 领域，日欧美企业（如信越、巴斯夫、东丽、SABIC）握有80%以上的产能，几乎无新增明确扩产计划。
-   **缺口**：传统燃料油过剩，高端化工品严重短缺。**这个结构性缺口在扩大**。在石脑油裂解乙烯这个层级，中国近几年投了大量的进口产能，这个环节已经消除了瓶颈。真正的瓶颈上移到**“如何用同样的基础原料，生产出价值高100倍的特种化学品”**这个层级。

## 第三层：FE 基本面验证

### 1. 天赐材料 (002709)

- **F（基本面）**：全球电解液龙头，市占率30%+，技术路线（LiFSI添加剂）领先。客户绑死宁德时代、LG、特斯拉。护城河在于**一体化**——从六氟磷酸锂（核心原料）到电解液，自己全包。而且它在扩产**LiFSI**，这是下一代高压快充电池的必需品。**基本面硬得不能再硬。**
  **评分：9/10**
- **E（预期差）**：市场还在用“周期股”的眼光看它，觉得锂价暴跌会杀利润。**大错特错**。锂价跌，电解液价格跌，但天赐的利润率在上升，因为它的原料成本降得更快。市场没看到的是，**电解液不是一个简单的周期品，它是一个技术壁垒极高的精细化工品。** 随着电池能量密度要求提高，配方升级，天赐的技术溢价会越来越高。预期差巨大。
  **评分：9/10**
- **V（估值）**：PE 34倍，在化工股里不便宜。但它是**高成长**的化工股，未来三年利润复合增速能看到30%+。PEG接近1，合理偏低。现在市场把它当周期股给估值，如果市场修正预期给成长股估值，空间很大。
  **评分：7/10**
- **FEV 总分**：F(9/10) + E(9/10) + V(7/10) = **25/30**

### 2. 福斯特 (603806)

- **F（基本面）**：光伏胶膜全球绝对龙头，市占率60%+。这个东西是光伏组件的“血液”，没了它，组件没法封装，透光率、寿命全完蛋。技术壁垒极高，新玩家3年都搞不定配方。福斯特的护城河在于**规模和成本**——它的采购价格比竞争对手低15%，利润率碾压。
  **评分：8/10**
- **E（预期差）**：市场认为光伏行业产能过剩，胶膜也会跟着降价。**部分正确，但逻辑有缺陷。** 行业过剩的是组件，不是胶膜。胶膜产能扩张门槛极高，福斯特的技术和成本优势，让它在行业洗牌时能吃掉更多份额。而且，**N型电池（TOPCON/HJT）对胶膜的要求更高、价值量更大**，福斯特是最大受益者。市场对“光伏过剩”的恐惧，掩盖了“龙头份额扩张”的现实。
  **评分：8/10**
- **V（估值）**：PE 62倍。**卧槽，这估值是认真的吗？** 在A股，这是给成长股的最高溢价之一。但别忘了，它也是周期股。现在市场把它当“周期顶峰”来定价，一旦行业出清，利润增速下滑，这个估值就是**泡沫**。买它，你是在赌它能用份额增长消化高估值。风险极高。
  **评分：3/10**
- **FEV 总分**：F(8/10) + E(8/10) + V(3/10) = **19/30**

### 3. 荣盛石化 (002493)

- **F（基本面）**：原油炼化 + PX（化工原料）。**这是我们这次分析里，最接近“卡脖子”上游的标的。** 中国PX国产替代的核心玩家，浙石化4000万吨炼化一体化项目是全球级的。但问题在于，**炼化是典型的周期性重资产**。它的利润高度依赖原油价格和行业开工率。基本面随着油价和政策起舞，不是垄断性的“卖水人”，而是“搬运工”。
  **评分：6/10**
- **E（预期差）**：市场认为化工周期在底部，等复苏。**这不算预期差，这是共识。** 真正的预期差在于，荣盛能否通过向下游延伸（如EVA、光伏材料），摆脱纯炼化的周期波动。目前看转型有，但进度不够快。预期差一般。
  **评分：5/10**
- **V（估值）**：PE 36倍，PB 2.4倍。对于一家纯周期炼化龙头，现在估值处于历史中位数，不贵不便宜。市场已经为“周期底部”定价了。想赚钱，必须等下一个周期起飞。
  **评分：6/10**
- **FEV 总分**：F(6/10) + E(5/10) + V(6/10) = **17/30**

### 4. 卫星化学 (002648)

- **F（基本面）**：**这里唯一的“价值陷阱”观察对象。** 它做的是**轻烃化工**，主要原料是乙烷、丙烷，技术路线独特，成本比石脑油路线低。它算是“卡脖子”下的受益者——因为中国的轻烃资源极度依赖美国进口（受益于中美贸易协定）。但它的核心逻辑是**扩产**，不是**不可替代性**。它的产品下游覆盖面广，但任何一个单一产品的壁垒都不高。护城河是成本和规模，不是技术。
  **评分：7/10**
- **E（预期差）**：市场把它当“成长化工”来买。**风险就在这里。** 它的增长全靠新项目投产（连云港C2、C3）。一旦项目投产不及预期，或者下游需求萎靡，估值会双杀。而且它的**原材料**完全依赖美国进口（乙烷），地缘政治风险极高。2022年俄乌冲突时，天然气价格暴涨直接干崩它的成本优势。市场似乎忽略了这一点。
  **评分：4/10**
- **V（估值）**：PE 13.8倍。看起来便宜，PB 2.2倍。但低PE是因为市场担心它的增长可持续性。**在卡脖子逻辑下，给一个依赖单一海外原料的公司13倍PE，我不觉得便宜，我觉得合理。** 买它，就是买一个高杠杆扩产且依赖海外原料的化工股。
  **评分：5/10**
- **FEV 总分**：F(7/10) + E(4/10) + V(5/10) = **16/30**
