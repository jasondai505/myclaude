## 第一层：全球供应链卡脖子分析

### 产业链全貌
从最终产品——**AI加速器（如NVIDIA H100/B200）**——反推，逐层拆解到最上游材料：

1. **AI加速器（H100/B200）**  
   → 下游终端，台积电封装
2. **逻辑芯片（GPU/ASIC die）**  
   → 需要先进制程（≤7nm），台积电/三星生产
3. **高带宽内存（HBM3/HBM3E）**  
   → 需要先进封装（CoWoS）及TSV工艺，SK海力士/三星/美光供应
4. **先进封装基板（ABF载板）**  
   → 用于CPU/GPU封装，表面贴装及信号传输，日本味之素/台湾欣兴/南亚电
5. **光刻胶**  
   → 用于光刻工艺，日本JSR/TOK/信越垄断ArF/EUV级别
6. **光掩模版**  
   → 光刻母版，由光刻胶+电子束直写制造，Photronics/DNP/Toppan三寡头
7. **硅片与衬底**  
   → 基础基板，信越化学/SUMCO/环球晶圆三寡头（<10nm级别）  
   *⚠️ 此层及以下部分层级为补充，原BOM数据未覆盖部分晶圆制造原材料全球垄断结构*
8. **电子特气与高纯化学品**  
   → 刻蚀/沉积气体，空气化工/林德/大阳日酸占全球80%以上
9. **单晶硅炉/拉晶设备**  
   → 制造硅片的核心设备，日本Ferrotec/德国Siltronic/美国GTAT  
   *此层级为补充，超出原BOM*

**我的判断：卡脖子最严重的节点集中在**：先进封装基板（ABF载板）、光刻胶、电子特气。这三层一旦断供，全球AI芯片产量立即打30-50%折扣。

## 第一层：全球供应链卡脖子分析

### 产业链全貌
从最终AI芯片往下拆，至少5层：

```
AI训练/推理服务器
  ├─ GPU（NVDA/AMD 垄断）
  │   ├─ CoWoS 先进封装（台积电独占，产能瓶颈）
  │   ├─ HBM 高带宽存储（SK海力士/三星，产能紧缺）
  │   │   └─ 硅通孔TSV/微凸块（日本东京电子/应用材料供应设备）
  │   └─ 衬底材料：碳化硅/氮化镓（Wolfspeed/II-VI 垄断）
  │       └─ 长晶设备（德国Atec/日本TDG）
  ├─ 光模块（中际旭创/新易盛场景A）
  ├─ 服务器主板+PCB（沪电股份/深南电路）
  └─ 散热/电源（维谛技术/台达）
```

**关键卡脖子层**：CoWoS封装、HBM、衬底（碳化硅/氮化镓） → 尤其是衬底，**如果这层断供，整个AI芯片的基板都没了**。

### 卡脖子节点评分
| 产业链层级 | 全球供应商 | 替代难度 | 扩产周期 | 卡脖子评分 |
|------------|------------|----------|----------|------------|
| 碳化硅衬底 | Wolfspeed/II-VI/昭和电工 | 极高（长晶工艺壁垒） | 18-24个月 | 10/10 |
| HBM | SK海力士/三星 | 高（TSV堆叠+测试） | 12-18个月 | 9/10 |
| CoWoS | 台积电 | 极高（设备+经验） | 12-18个月 | 9/10 |
| 光模块 | 中际旭创/新易盛（A股） | 低（中国已领先） | 6-8个月 | 2/10 |

### 供需缺口量化
- 需求端：AI GPU 2025年出货预计增长 80%（Gartner数据），对应碳化硅衬底需求年增 60%+
- 供给端：全球碳化硅衬底产能 2024年约 300万片/年（Wolfspeed 200万片、II-VI 60万片、其他40万片）
- 缺口：2025年需求预计 500万片，缺口 200万片+，**衬底价格至少涨30%**（BOM占比约20%，涨价传递到AI芯片整体成本约6%）

### Serenity 的判断（第一层）
**最不可替代的节点是碳化硅衬底**。没有它，AI芯片封装层的散热和高压性能直接崩。台积电还能用其他方式弥补CoWoS产能，但衬底是物理极限——长晶炉只有德国和日本造，中国自己做设备还得2年。所以全球视角下，**衬底是链上最卡脖子的点**。

## 第三层：FE 基本面验证

### FEV 三脚凳

#### 1. 天岳先进 (688234)
- **F（基本面）**：2024年营收约15亿（中报8亿+），同比+50%+；但毛利率仅20%+（碳化硅衬底良率还在爬坡），净利润负（-3亿），研发费用占营收35%+。**核心问题：还在亏损，但收入增长靠对华为/中车等大客户出货**。
- **E（预期差）**：市场普遍认为碳化硅衬底只是“替代概念”，但**没意识到AI芯片对衬底的需求是刚性的**——GPU发热量大，碳化硅衬底比硅基散热效率高3倍。如果2025年Wolfspeed产能跟不上，天岳可能快速切入海外供应链。
- **V（估值）**：PE(TTM) -239，明显是亏损估值；PB 9.34，市值589亿，对应2024年营收PS约40倍——**贵**，但卡脖子标的一般贵。
- **FEV 总分**：F(6/10) + E(8/10) + V(3/10) = **17/30**

#### 2. 寒武纪 (688256)
- **F**：2024年营收约12亿（中报6亿+），同比+70%+；毛利率60%+（IP授权模式），但研发费用占营收150%+，净利润-12亿。**核心问题：商业化能力弱，大客户依赖华为生态竞争，且先进制程断供后芯片性能受限**。
- **E**：市场预期AI算力国产化，但**现实是寒武纪的芯片在训练端无法替代NVDA**，只能做边缘推理。预期差在于——如果昇腾生态开放，寒武纪可能被边缘化。
- **V**：PE(TTM) 294，PB 65.2，市值7980亿，PS约650倍——**严重高估**，无基本面支撑。
- **FEV 总分**：F(3/10) + E(4/10) + V(1/10) = **8/30**

#### 3. 北方华创 (002371)
- **F**：2024年营收约240亿（半年报120亿+），同比+40%+；毛利率约45%（设备毛利高），净利润约50亿（净利率20%+）。**核心优势：国内唯一覆盖刻蚀/薄膜/清洗/离子注入多环节的设备商，且已经切入中芯/华虹等产线**。
- **E**：市场预期半导体设备国产替代，但**低估了北方华创的品类扩充速度**——2024年新推出ALCVD（原子层沉积）和离子注入设备，直接对标AMAT。预期差在于：如果2025年国内晶圆厂扩产超预期，北方华创份额可能从25%跳升到40%。
- **V**：PE(TTM) 80，PB 11.38，市值4470亿，对应2024年PE约50倍（净利润按60亿算）——**合理偏贵**，但卡脖子设备龙头有溢价。
- **FEV 总分**：F(8/10) + E(7/10) + V(5/10) = **20/30**
