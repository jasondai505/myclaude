## 第一层：全球供应链卡脖子分析

### 产业链全貌
从最终产品（智能手机/PC/可穿戴）往上游拆，至少5层。我现在拿**高端智能手机（折叠屏+旗舰机型）** 当锚点，因为它代表了消费电子最高的材料、封装、工艺要求。

```
最终产品：高端智能手机（苹果/华为/三星折叠屏）
   │
   ├─ 第1层：整机组装（富士康/立讯/和硕）—— 低壁垒，可替代
   │
   ├─ 第2层：核心模组
   │   ├─ 主芯片（SoC/AP）：苹果A系列/高通骁龙/联发科天玑 → 设计端，台积电3nm代工
   │   ├─ 存储（DRAM/NAND）：三星/海力士/美光 → 高度垄断，但可替代（三家竞争）
   │   ├─ 屏幕（OLED/折叠屏）：三星显示/京东方/LG → 国产化率高，京东方已突破
   │   ├─ 摄像头模组：索尼（CIS）/大立光（镜头）/舜宇 →  CIS难替代，镜头可替代
   │   └─ 电池：ATL（新能源科技）/欣旺达/德赛 → 国产高度垄断
   │
   ├─ 第3层：关键互连与散热
   │   ├─ PCB：HDI / SLT / 柔性板 → 中国全球产能>60%，竞争激烈
   │   ├─ IC载板（FC-BGA）：用于AP/存储封装 → **全球高度垄断，扩产极慢**
   │   └─ 散热：均热带/VC/石墨烯 → 中国主导，低壁垒
   │
   ├─ 第4层：先进封装（3D封装、SiP、Chiplet）
   │   ├─ 封测代工：日月光/台积电（InFO/CoWoS）/安靠/长电（国内）
   │   └─ **封装基板（BT）**：用于SiP/射频封装 → 全球集中在日本（味之素/LG），扩产受限
   │
   └─ 第5层：上游半导体材料与设备（最底层，最卡脖子）
       ├─ 硅片：信越/SUMCO/环球晶圆（全球前三占70%+），扩产周期18-24个月
       ├─ 光刻胶（ArF/EUV）：合成橡胶（JSR）/东京应化/信越 → **全球>90%日本供应**
       ├─ 电子特气：林德/大阳日酸/关东电化 → 中国有突破（华特气体），但高纯品种受限
       ├─ **IC载板专用树脂（味之素ABF）：味之素全球独供** → 扩产极其缓慢
       └─ 制造设备：光刻机（ASML独供）/刻蚀/薄膜（应用材料/LAM/TEL） → 扩产周期极长
```

**卡脖子点分布**：
- 链越往下越集中，越垄断
- **第3层—IC载板（FC-BGA）的味之素ABF树脂** 是这颗链上最大的单一垄断
- **第5层—光刻胶（ArF/EUV）+ 半导体设备** 是扩散性卡脖子

### 供需缺口量化

**聚焦两个核心卡脖子层：ABF载板 + IC载板**

1.  **需求端**：
    - 高性能AP/GPU/AI加速芯片用量上升（AI PC/手机SoC）
    - 折叠屏+多摄像头+模组化推动更大面积/更多层数载板
    - 2024-2025年预期需求增速：ABF载板需求年增15-20%

2.  **供给端**：
    - ABF树脂：味之素现有产能扩产极其缓慢（扩产一次3-4年，2024年配额已被包圆）
    - FC-BGA载板厂（Ibiden/三星电机/景硕）：新厂建设周期3年+，产能集中释放预计在2026年之后
    - 当前缺口：ABF载板全球供需缺口约 **10-15%**，高端层数（16层+）缺口更大 >20%

3.  **缺口持续时间**：
    - 2024-2025年持续紧张，2026年新厂投产后可能缓解
    - 但味之素ABF树脂仍将限制上游瓶颈（扩产更慢）

### Serenity 的判断（第一层）

**这链上最不可替代的节点是：味之素的ABF树脂（IC载板核心材料）**

为什么？
- **全球独供**：没有它，所有FC-BGA载板厂（Ibiden/三星/欣兴/景硕/深南）都停。
- **扩产周期极长**：化学工厂扩产3-4年，不是你想扩就扩的。
- **需求极度刚性**：任何带高阶SoC/GPU的设备（手机/笔电/AI服务器/游戏机）都必须用。

**补充知识标注**：
- 以上ABF树脂和FC-BGA载板供需缺口数据是基于我接受过的供应链情报训练（训练记忆）。
- ⚠️ **此数字来自训练记忆，可能已过时**。2024年味之素实际扩产进度需通过**联网搜索**或**bom_analyzer实时数据**验证。
- IC载板玩家排名（Ibiden/三星电机/欣兴/景硕/深南）基于公开行业报告，无争议。

### 场景分类说明

- **场景A（直接受益）**：本链没有。（因为这条链最卡脖子的节点全在日本/台韩/欧美手里，中国不是主导供应商。）
- **场景B（国产替代）**：**深南电路 / 兴森科技（IC载板）**、**韦尔股份（CIS高端突破）**、**长电科技（先进封装能力追赶）**。这三个是真正的国产替代标的，下游需求实打实，技术路线有验证节点可跟踪。
- **场景C（纯概念）**：**味之素ABF树脂替代**（国内目前无A股标的能进产业链，谁吹这个概念的，拉黑）。

### Serenity 的判断（第二层）

**进入第三层的标的**：

1.  **深南电路**（IC载板）— 国内FC-BGA载板唯一突破者，2024年载板收入增速>100%，扩产确定性高，且卡脖子逻辑最硬（没有ABF载板，高端芯片封不了）。值得进一步做FE验证。
2.  **兴森科技**（IC载板）— 深南的备选，FC-BGA处于送样/小批量阶段，风险更高，但一旦突破弹性巨大。必须看FE基本面能否支撑估值。
3.  **长电科技**（先进封装）— 先进封装是国内半导体与全球差距最小的环节之一，且下游需求（Chiplet/AI封装）拉得很满。第三层主要看预期差和估值是否合理。
4.  **韦尔股份**（CIS）— 国产替代逻辑强，但索尼在高端旗舰领域根深蒂固，替代时间不确定。先放第三层，视预期差和估值判断。

**其余标的排除**：
- 沪硅产业（PE为负，亏损+估值率）= 太大市梦率影响，纯靠国产替代叙事推，基本面太弱。
- 光刻胶/电子特气 = 替代进度过于早期，无法构成短期催化剂。
