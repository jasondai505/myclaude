# Serenity · 产业链卡脖子分析方法论

> 源自 Serenity（前 Reddit WSB 散户 → AI 半导体供应链分析师），本土化为 A 股三层分析法。
> 覆盖 35 个产业链，引擎代码 `engine_serenity.py` + `serenity_kb.py`。

---

## 一、核心思想：不买最大的，买最不可替代的

从整条供应链出发，找到「如果这层断了，全球产业链停」的节点。不是选股 → 验证，而是反推供应链 → 找瓶颈 → 映射标的 → 验证。

## 二、三层分析框架

```
第一层：全球供应链卡脖子分析（Global）
  从最终产品反推供应链，逐层分析替代难度/扩产周期/供需缺口
  → 找到最不可替代的卡脖子节点

第二层：A 股映射（Mapping）
  场景A（直接受益）：中国是这层的关键供应商 → A 股直接受益
  场景B（国产替代）：海外垄断 + 国内在追赶 → 国产替代标的
  场景C（纯概念）：没有中国标的 → 不做

第三层：FE 基本面验证（Validation）
  FEV 三脚凳（基本面/预期差/估值）+ Focus Five
  → 只有三层全过的才是可操作标的
```

## 三、卡脖子三要素

| 要素 | 含义 | A 股特别关注 |
|------|------|------------|
| 供给无法快速扩产 | 工厂建设周期长、技术壁垒高、原材料稀缺 | 国内产能瓶颈 + 海外设备禁运叠加 |
| 需求刚性 | 下游必须用它，无替代路线 | 新能源/半导体/军工等政策刚性方向 |
| 关注度低 | 大资金还没进，估值还没打满 | A 股题材热度尚未扩散到上游 |

**关注度的双向信号**：低关注度 = 买入信号；极端一致性共识 = 反向信号（逻辑已充分定价）。

## 四、卡脖子层子类型拆分

不能停在「层」这个级别。同一层内不同子类型的供需失衡程度可能差一个数量级。

例：光通信中「激光器」是一个卡脖子层，拆进去有 EML（用于可插拔收发器）和 CW（用于 CPO）——不同供应商、不同扩产周期、不同紧缺程度。

**规则**：高评分卡脖子层必须再拆一层，A 股映射要分清楚标的对应哪个子类型。

## 五、主题生命周期

| 状态 | 特征 | 行动 |
|------|------|------|
| 逻辑成立 | 供需缺口已验证，但无明确时间表 | 跟踪 |
| 催化剂临近 | 送样/认证/量产 1-2 季度内 | 关注 |
| 价格未充分反映 | 市场还没意识到 | 可行动 |

催化剂 > 2 季度看不到的 → 降仓位或只跟踪，即使逻辑完全正确。

## 六、需求验证信号（领先指标）

不等财报，用以下信号做前瞻验证：

1. **LTA（多年长期协议）签署**：大客户锁产能 = 需求真实 + 供给紧缺被证实
2. **上游设备采购订单加速**：设备订单的频次和规模领先于产能释放
3. **竞争性锁仓**：一家签完 LTA 后竞争对手恐慌跟进 → 需求螺旋

## 七、垂直整合潜力

同一卡脖子节点，不同公司天花板取决于能在价值链上爬多高：

```
只卖原材料 < 卖中间产品 < 卖完整模块 < 卖系统级产品
每往上一层，TAM 和利润率跳一个台阶
```

## 八、评分体系

| 维度 | 评分 | 说明 |
|------|:--:|------|
| 卡脖子分（global_chokepoint_score） | 0-10 | 全球供应链替代难度量化 |
| F 基本面分 | 0-10 | 盈利质量 + 竞争壁垒 |
| E 预期差分 | 0-10 | 市场认知偏差 + 边际变化 |
| V 估值分 | 0-10 | PE/PB 相对历史/行业分位 |
| FEV 总分 | 0-30 | 三脚凳综合，≥20 才进入候选 |
