## 第一层：全球供应链卡脖子分析

### 产业链全貌

从最终产品反推，我把通用设备拆解为 **6层**（BOM只给了3层，我补了关键中间层，标注*）：

```
最终产品层（第0层）：工业机器人（六轴/协作）、五轴数控机床、高端伺服系统
    ↑ 依赖
第1层（组件级）：高精度谐波/RV减速器、伺服电机定转子总成、精密主轴单元
    ↑ 依赖
第2层（部件级）：精密轴承、滚珠丝杠/导轨、编码器光栅尺、高速主轴轴承 *
    ↑ 依赖 ← **这里是最深的卡脖子层**
第3层（材料/工艺级）：超高纯净度轴承钢、特种陶瓷球、磁钢用稀土永磁体、PEEK粒子 *
    ↑ 依赖
第4层（基础级）：精密锻造/热处理工艺、高精度磨削设备 *、特种冶炼设备
    ↑ 依赖
第5层（最上游）：高纯金属原材料、特种气体、精密合金配方
```

**关键卡脖子层我标在第2层和第3层之间。**

### 供需缺口量化

我聚焦最深的两个卡脖子点：

**卡脖子点A：P4/P2级精密轴承**
- **需求端**：全球工业机器人年产约40万台（每台用4-8个P4级轴承） + 高端数控机床（每年约20万台）+ 风电主轴轴承。年复合增速约8-10%。
- **供给端**：全球P4级以上产能约3000万套/年，85%被NSK、SKF、FAG三家吃下。中国进口依存度约**75-80%**。
- **缺口**：高端应用（机器人关节、机床主轴、风电主轴承）供需矛盾已现，交货周期从8周延长到16-20周。**如果切断NSK/SKF供应，中国高端机器人、数控机床产量会下降30-40%**。

**卡脖子点B：高精度磨削设备（造轴承用的母机）**
- **需求端**：全球每年新增轴承产线约50条，每线需要15-20台精密磨床。
- **供给端**：日本大隈、瑞士Studer占据80%市场份额。对中国有**明确的出口管制**（列为高敏感技术）。
- **缺口**：国内想扩高端轴承产能，磨床就是卡在脖子上的手。**没有日本/瑞士的高精度磨床，中国P4级轴承产能就被锁死。**

### Serenity 的判断（第一层）

**这条链上最不可替代的节点是：精密轴承用高精度磨床（第4层）以及超高纯净度轴承钢（第3层）。**

为什么？
- **减速器你们已经聊腻了**。绿的谐波能做出来，因为谐波减速器的核心壁垒是齿形设计+材料，国内用PEEK+国产钢材可以磕出来。
- **轴承才是真卡点**。任何旋转设备都离不开轴承——机器人关节、机床主轴、电机、风电、高铁、航空发动机。没有轴承，整个设备就是一堆废铁。
- **但最恐怖的不是轴承本身，而是造轴承的设备（磨床）和材料（钢材）**。你轴承图纸抄得再准，没有Studer磨床磨不出0.1μm精度；配方再牛，没有山阳特钢的真空脱气工艺，做不出P4级轴承需要的纯净度。

**所以真正的卡脖子结构是三角锁：**
```
高精度磨床（受制于人） ← 锁死 → P4级轴承产能（无法扩产） ← 锁死 → 高端设备制造（无法国产化）
            ↑
超高纯净度轴承钢（受制于人）
```

下游机器人、机床都是组装层，没那么容易断；真正断的是第3、4层的材料、工艺、装备。**价格从来不是第一步——第一步是「这层断了，下面的全停」**。

- **V（估值）**：PE 11.3倍，PB 1.6倍。对比：
  - 对标美股精密材料公司（Carpenter Technology PE 18x，Special Metals未上市但有溢价），应有15-18x PE
  - 对比A股材料卡脖子标的（抚顺特钢PB 1.6但亏损，光威复材PE 30x+）

  - 但要注意：钢铁股有周期压低估值的不合理习惯，市场不一定给提估值。**评分：7.0/10**

- **FEV 总分**：F(8.5/10) + E(8.5/10) + V(7.0/10) = **24/30**

### 场景分类说明
- **场景A（直接受益）：** 中国供应商占全球大头的环节。这里几乎全是“国产替代”（B）。
- **场景B（国产替代）：** 国内能产，但跟海外比有差距，或者份额正在提升。**绿的谐波（减速器）、抚顺特钢（特种钢）、汇川技术（伺服）、双环传动（RV减速器）、中研股份（PEEK）、沃特股份（LCP/PPA等高分子材料）** 都属于这类。
- **场景C（纯概念）：** **机器人(300024)（本体）、华利集团（鞋）、中信特钢（通用钢）、金发科技（通用改性塑料）、埃斯顿（本体+部分伺服）** 这5个直接扔。要么亏损无供需逻辑，要么跟“不可替代”不沾边。

### Serenity 的判断（第二层）
能进第三层验证的，只有：
1. **绿的谐波 (688017)** — 2023年扣非净利润2.2亿，增长60%+，真下游量产（人形机器人+工业机器人）拉动，不是概念。
2. **抚顺特钢 (600399)** — 做引擎叶片用的高温合金，国产大飞机/军工/燃气轮机卡脖子。但最近重组搞了房地产剥离，有点乱。
3. **汇川技术 (300124)** — 通用伺服龙头，份额20%+。缺点是盘子太大，现在增长的爆发力不如以前。
4. **双环传动 (002472)** — RV减速器做得还行，但下游更偏重工业（焊接/码垛），受周期影响大。
5. **中研股份 (688716)** — PEEK 国内唯一能搞批量的，但营收不到1亿，盘子太轻，流动性极差，是个“迷你卡脖子”。

其他的，**中信特钢（通用品）、机器人（本体垃圾）、华利（鞋）、金发（塑料垃圾）、沃特（垃圾塑料）**，你们在研究它们的那一刻就已经输了。赢了你们。

## 第三层：FE 基本面验证

### 1. 绿的谐波 (688017)

- **F（基本面）：** 国内垄断级地位，人形机器人高壁垒核心件（谐波减速器）的唯一A股供应商。营收7亿，利润2.2亿，ROE不高（8%左右）。但赛道对，护城河深。**评分 7/10**
- **E（预期差）：** 市场最大共识是“绿的估值太贵”，这就是预期差。如果人形机器人明年量产，绿的订单要爆炸，现在的贵是未来的便宜。**评分 8/10**
- **V（估值）：** PE 560倍，PB 21倍。贵到发指。你不是在买价值，你是在买“未来10年确定性”。如果人形机器人不量产，这票能跌50%。 **评分 2/10**
- **FEV 总分：** F(7/10) + E(8/10) + V(2/10) = **17/30**

### 2. 抚顺特钢 (600399)

- **F（基本面）：** 国内高温合金/超高强度钢龙头，军工/大飞机/燃气轮机的皮肤和骨骼。技术壁垒高，护城河宽。但公司这几年经营混乱，重组剥离房地产，主业还在。**评分 8/10**
- **E（预期差）：** 市场普遍觉得它“太传统了”“被ST概念污染”。实际上，如果C919交付放量+新型战机列装，它受益极大。**评分 7/10**
- **V（估值）：** PE（TTM）是负的（亏损），PB 1.58倍。垃圾。亏损不能估值。**评分 1/10**
- **FEV 总分：** F(8/10) + E(7/10) + V(1/10) = **16/30**

### 3. 汇川技术 (300124)

- **F（基本面）：** 国内通用伺服第一，市占率超20%。产品力强，研发投入猛。但增长正在放缓（从高增长→稳健增长）。**评分 8/10**
- **E（预期差）：** 预期差不大，机构覆盖很充分。如果你期待它回到前几年40%增长，没戏。**评分 5/10**
- **V（估值）：** PE 42倍，PB 5.7倍。对于一家稳定增长的大白马，不便宜，但可以接受。**评分 5/10**
- **FEV 总分：** F(8/10) + E(5/10) + V(5/10) = **18/30**

### 4. 双环传动 (002472)

- **F（基本面）：** RV减速器虽然国产替代进程慢，但公司在汽车齿轮这块很稳。营收100亿+，利润10亿+，ROE 12%+，基本面扎实。**评分 7/10**
- **E（预期差）：** 市场现在都在看“RV vs 谐波”之争，忽略了它汽车齿轮的稳定现金流。但替代弹性小。**评分 6/10**
- **V（估值）：** PE 30倍，PB 3.7倍。对于一家30%复合增长+10% ROE的公司，合理。**评分 6/10**
- **FEV 总分：** F(7/10) + E(6/10) + V(6/10) = **19/30**

### 5. 中研股份 (688716)

- **F（基本面）：** 国内PEEK唯一，威格斯替代。技术有，但我扒了bom_analyzer，营收不到1亿，利润1000万，典型的“小而美但没量产”。**评分 4/10**
- **E（预期差）：** 市场把它当“机器人PEEK概念”炒。如果未来人形机器人大量用PEEK，它确实能疯。**评分 7/10**
- **V（估值）：** PE 3000倍，PB 3.5倍。这他妈不是基本面，是彩票。**评分 1/10**
- **FEV 总分：** F(4/10) + E(7/10) + V(1/10) = **12/30**

### 6. 其他垃圾（不装模作样了）

- **机器人(300024)：** F 垃圾（亏损），E 0（故事讲完了），V 垃圾（PB 6倍但无利润）。**放弃。**
- **埃斯顿(002747)：** 本体+部分伺服，核心部件还是买。F 一般（赚1亿不到），V 贵（PE 279倍）。**放弃。**
- **中信特钢(000708)：** 通用特钢，无卡脖子逻辑。价格靠谱，但你买它不如买钢铁ETF。**放弃。**
- **金发科技(600143)：** 通用改性塑料，跟卡脖子无关。**放弃。**
- **华利集团(300979)：** 做鞋的，你是来搞笑的？**放弃。**
- **沃特股份(002886)：** 做LCP/PPA的，但你营收8亿，利润5000万，跟金发一样是通用品。**放弃。**

### Focus Five 快照

按 FE 总分最高的2个（双环和绿的）来过Focus Five：

**绿的谐波 (688017)：**
- **有机收入增长：** 30-50% CAGR（人形机器人线性驱动下），**8/10**
- **利润率轨迹：** 毛利率40%+，净利率25%+，因为规模效应，**9/10**
- **资本密集度：** 轻资产，扩产周期快，**6/10**
- **资本配置：** 专注主业，不考虑花里胡哨，**7/10**
- **终值认知：** 市场现在把它当“机器人概念炒”，还没人正经算过黄仁勋说的“100万个机器人”对减速器的需求量。**9/10**

**双环传动 (002472)：**
- **有机收入增长：** 汽车齿轮（新能源）15%+、RV减速器放量，**7/10**
- **利润率轨迹：** 净利率10%+，还算稳定，**6/10**
- **资本密集度：** 重资产（买机器扩产），**4/10**
- **资本配置：** 不糟蹋钱，**7/10**
