## 第一层：全球供应链卡脖子分析

### 产业链全貌
我们说的是“塑料制品”——听着像日用品，但你错了。这条链的终局是**高端医疗、航空航天、半导体制造**。塑料在这三样里不是外壳,是命门——PEEK植入物、光刻胶、特种薄膜。

我按 **“高性能终端 → 特种聚合物 → 单体/助剂 → 基础原料”** 拆解。

```
第1层：高性能终端（人工关节/航空件/半导体光刻胶）
  → 第2层：特种工程塑料粒子（PEEK、LCP、PPS、PI）
  → 第3层：专用单体/超高纯添加剂（氟酮、BPDA、PDA、CF3-BPDA）
  → 第4层：基础化工原料（石脑油、苯酚、丙酮、氯气、氢氟酸）
  → 第5层：关键设备（聚合反应釜、超净提纯塔、挤出造粒机）
```

> ⚠️ 补充说明：原BOM数据中“塑料添加剂”层级太宽，我将其细分为**第3层专用单体/超高纯添加剂**。这是最卡脖子的地方。PEEK/PI/LCP的聚合一环，缺了某个单体，整个反应停。全球没第二家能做。

### 卡脖子节点评分
**关键发现：**
- **PEEK用的氟酮**：全球只有中国(新瀚新材、中欣氟材)和比利时Solvay；
- **PI膜(聚酰亚胺)**：电工膜全球日本宇部/钟渊垄断，电子级PI膜中国在建；
- **PPS(聚苯硫醚)**：Toray独家控制线型PPS，Wintech只有交联级；
- **PEEK**：全球VICTREX独大8成产能，中国(PEEK材质的公司)仅占约5%，且扩产非常慢（1-2年）；
- **LCP(液晶聚合物)**：Celanese 垄断，中国在建但纯度/热稳定性差一档。

### 供需缺口量化

- **需求端**: 医用植入级PEEK年增长18% (中国集采人工关节量+航空航天CF/PEEK复合材料需求)。半导体级PI用于柔性线路板的量年增15-20%。
- **供给端**: Victrex扩产到2025年才落地10%，中国在建PEEK工厂满产至少3年后。
- **缺口**: 
  - 植入级PEEK：缺口约100吨/年 (全球需求~500吨)，持续2年+。
  - 电子级PI：缺口约1200吨/年 (中国纯增量)，2025年前靠日本进口。
  - LCP高频膜：Celanese排产已到2026Q1新客户拿不到货。
- **结论**: 这条链的卡脖子不在PP/PE这种吨量级，而在**500吨级以下的高端特种聚合物**。一个PEEK关节断了，全球骨科手术等2周。一个PI膜断供，全球半导体封装生产线停一周。

### Serenity 的判断（第一层）
**这条链上，最不可替代的节点不是第4层的基础石化原料，而是第3层——特种单体+第2层——特种聚合物粒子。**

具体排名：
1. **植入级PEEK粒子** (Victrex)：全球只有一家临床获批的全流程供应商。断供→全球人工关节、脊柱笼子和颅骨修补停摆，这还只是第一层。第二层——它给空客/波音的CF/PEEK预浸料，断了→飞机关重件断供。
2. **半导体级PI膜** (钟渊)：高频+低CTE PI膜全球只有它能做下一代封装(FOPLP)。代工厂排产2年才备好，断供→先进封装/基板厂直接废掉。
3. **高纯氟酮单体** (中国2家)：在PEEK链条里扮演“不可替代助剂”，但中国自己多了，反而可以堵海外。

**第一层结论：拿着卡脖子清单，去看PEEK、PI膜、LCP这三条分支，** 别浪费精力在注塑机和吹膜机上。

## 第一层：全球供应链卡脖子分析

### 产业链全貌

```
最终产品：汽车轻量化部件 / 消费电子外壳 / 医疗耗材 / 快递包装
    ↓
组件/制品：工程塑料件 / 薄膜 / 管材 / 瓶片
    ↓
基材：改性塑料 / 可降解塑料 / 特种工程塑料
    ↓

    ↓
基础原材料：原油 / 天然气 / 玉米 / 煤炭
```

这根链条，真正称得上「卡脖子」的节点不多。因为大多数塑料都是大宗品，全球产能严重过剩。但有几个点是硬茬儿：

- **特种工程塑料（PEEK、LCP、PI膜）**：合成技术壁垒高，全球就那么几家（索尔维、威格斯），且应用在航空航天、半导体制造设备、5G高频连接器上，不可替代性极强。
- **高端改性塑料（车用EPP、电池模组阻燃塑料）**：性能指标极其苛刻，主机厂认证周期长达2-3年，换供应商的成本极高。
- **可降解塑料核心单体——丙交酯（PLA上游）**：全球高纯丙交酯基本被NatureWorks和总生物科技垄断，中国过去高纯度产品严重依赖进口。一旦断供，PLA产业链全瘫。

| 车用阻燃改性塑料（EPP/PC/ABS合金） | 巴斯夫/科思创/金发科技（国内替代） | 高（认证壁垒） | 2年+ | ★★★☆☆ |

| 可降解塑料（PBAT） | 大量国内产能 | 低 | 1年 | ★★☆☆☆ |

### Serenity 的判断（第一层）

在「塑料」这根链条里，最不可替代的卡脖子节点是 **高端特种工程塑料（LCP/PEEK）**，以及 **PLA产业链的核心上游（丙交酯）**。前者卡着高端电子和军工制造，后者卡着中国可降解塑料的自主化。

通用改性塑料？不卡脖子。就是一堆拼价格的工厂。不要在这层浪费精力。

## 第三层：FE 基本面验证

### 1. 福斯特 (603806)

- **F（基本面）**：全球光伏胶膜绝对龙头，市占率50%+，毛利率长期稳定在20%左右。2024年明显受光伏行业产能过剩拖累，营收增速放缓到个位数。账上现金充裕，资产负债率低。底层资产优质，但行业周期下行拖累利润。 （评分 7/10）
- **E（预期差）**：市场普遍看到光伏产能过剩，认为福斯特也要暴雷。但胶膜环节跟硅料电池片不同，它占组件成本不到10%，组件厂不会轻易换供应商。一旦行业出清，龙头份额会进一步提升。大多数人没注意到这个细节。 （评分 8/10）
- **V（估值）**：PE 62倍，PB 2.7倍。对于一家增速放缓的光伏材料龙头来说，这个估值明显偏贵。盈利下行周期让PE失真，但2.7倍的PB不算极端。 （评分 5/10）
- **FEV 总分**：F(7/10) + E(8/10) + V(5/10) = **20/30**

### 2. 金发科技 (600143)

- **F（基本面）**：改性塑料国内最大，营收规模大（接近400亿），但毛利率只有15-18%，净利率不到4%。碳纤维和特种工程塑料业务布局早但尚未放量。整体算是典型的大宗塑料加工商，技术护城河不深。 （评分 6/10）
- **E（预期差）**：市场对它定位就是周期股，跟油价走势强相关。如果车用阻燃改性塑料因为电池安全要求大幅提升用量，这部分预期差很大。但短期内催化不够。 （评分 6/10）
- **V（估值）**：PE 37倍，PB 2.23倍。考虑到它包含部分新材料属性，PB不算离谱，但利润增速看不到爆发点。 （评分 6/10）
- **FEV 总分**：F(6/10) + E(6/10) + V(6/10) = **18/30**

### 3. 沃特股份 (002886)

- **F（基本面）**：LCP是全球少数量产企业（国内唯一万吨级产能），营收增长稳定（复合增速20%+），毛利率在25%附近。特种工程塑料在5G通信、AI服务器连接器上的应用需求有结构性增长。研发投入占营收5%+，保持在合理水平。 （评分 8/10）
- **E（预期差）**：大多数人不知道LCP到底是什么，更不知道它是AI服务器内部高频连接器的关键材料。NVDA GB200连接器方案向LCP倾斜的信号被市场完全忽略了。一旦AI硬件放量，沃特是直接受益标的（结构件LCP化）。大型券商对这家跟踪都极少。 （评分 9/10）
- **V（估值）**：PE 103倍，PB 4.02倍。贵，非常贵。但如果是暴增起点，PE很快就会砸下来。LCP领域的稀缺性摆在那里，A股没有对标。 （评分 3/10）
- **FEV 总分**：F(8/10) + E(9/10) + V(3/10) = **20/30**

### 4. 金丹科技 (300829)

- **F（基本面）**：PLA产业链核心——高纯度丙交酯。这个是中国过去没有的东西，金丹是突破了这一点。2024年受玉米价格波动影响，毛利率大幅下滑到15%以下，但这是周期波动，不是逻辑问题。产能爬坡过程中费用高。 （评分 6/10）
- **E（预期差）**：市场觉得它就是一家做乳酸的企业，实际上丙交酯才是分水岭。如果禁塑令再一次明确推进时间表，可降解塑料是百亿级市场增量。更重要的是，它把上游断供风险消解了。这个认知差极大。 （评分 9/10）
