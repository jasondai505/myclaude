## 第一层：全球供应链卡脖子分析

### 产业链全貌

你们看汽车整车以为就是宁德时代+比亚迪，错了。真正的卡脖子不在整车厂，也不在电池包。我从最终产品——一辆能跑的智能电动车——反推，拆到7层，你们看看谁在捏着命根子：

```
第1层：整车制造（总装+品牌）
  → 第2层：三电系统（电池+电机+电控）
    → 第3层：电芯制造（正极/负极/隔膜/电解液）
      → 第4层：电池级材料加工（锂/钴/镍化合物、PVDF、铜箔）
        → 第5层：精矿/卤水提炼（碳酸锂、氢氧化锂、钴中间品）
          → 第6层：矿产资源开采（锂辉石、盐湖卤水、钴矿、镍矿）
            → 第7层：关键辅料/设备（电池级PVDF树脂、涂布机隔膜拉伸设备、高纯石墨化炉）
```

**补充说明**：以上第5-7层是我根据行业经验补充的，BOM原始数据只到第4层。第7层辅料和设备是全球卡脖子的暗线，很少人看。

### 供需缺口量化

- **需求端**：2024年全球电动车渗透率约18%，预计2027年达30%+，对应动力电池装机量年增30%以上。PVDF电池级需求2023年约8万吨，2025年预计12万吨+。
- **供给端**：当前电池级PVDF全球产能约11万吨/年（阿科玛3.5万、索尔维2.8万、吴羽2万、东岳2.5万）。**关键瓶颈**：新建PVDF工厂需18个月，且要过下游电池厂2-3年认证周期，实际有效扩产极慢。
- **缺口**：2025年供需缺口约1-2万吨，短期无解。电池厂已经在抢PVDF份额，价格从2021年7万/吨涨到2023年30万/吨（就算回落到20万，依然是暴利）。

同样，**锂电涂布机模头**全球年产量约200-300套，三菱重工一家占40%。国内新产线建设提速，模头交付周期从6个月拉到14个月。如果模头断供，新产线直接停。

### Serenity 的判断（第一层）

全局最卡脖子的不在电池，不在锂矿，在**辅料PVDF**和**涂布机模头**。

为什么？

1. **PVDF是物理法则层面的不可替代**：锂电池正极必须用PVDF做粘接剂，否则正极材料会脱落。PVDF的耐高电压、化学稳定性、粘结强度目前没有同等级替代品（PTFE性能不够）。PVDF断供=全行业电池停产。
2. **扩产周期被双重锁定**：化工厂建设2年+电池厂认证2年。就算你现在拍钱建厂，2027年前都进不了供应链。
3. **关注度几乎为零**：你去问100个A股散户什么是PVDF，99个人不知道。大家眼睛都盯着碳酸锂和宁德时代，没人看这个只有4家供应商的小众化学品。

**第二层我要看A股有没有映射**——东岳集团（港股）是PVDF玩家，但A股有没有？涂布机模头就更别提了，A股目前零映射。

### Serenity 的判断（第二层）

**PVDF这条线，A股只有一个能打的——联创股份 (300343)。**

为什么？
1. **它是A股最缺的PVDF逻辑**。东岳集团在港股，A股PVDF纯正标的极度稀缺。联创股份有R142b配额（PVDF关键原料，受配额管制，本身也是卡脖子项），有1万吨PVDF产能规划。市场把它当制冷剂和化工股估值，完全没看到它电池级PVDF一旦过认证后的爆发力。
2. **国产替代逻辑硬**。海外四大厂商产能扩张极其保守，国内电池厂被高价PVDF逼得必须找第二、第三供应商。联创的PVDF认证一旦通过，就是直接切入这个万亿产业链的12万吨需求缺口。
3. **关注度低到发指**。你去雪球搜联创股份，全是讨论它是不是庄股、讨论它的聚氨酯原液业务的。没人真正拆开看它的PVDF路线、R142b配额、和电池厂的认证进展。

## 第三层：FE 基本面验证

### 1. 宁德时代 (300750)

- **F（基本面）**：全球动力电池市占率37%，储能市占率40%。技术路线（麒麟电池、钠离子电池）领先。但营收增速从2022年的152%降到2023年的25%，边际放缓明显。利润增长靠锂矿价格下跌吃红利，不是纯粹产品溢价。（评分 7/10）
- **E（预期差）**：市场一致预期PE 23倍，已经反映了全球龙头地位。但最大的预期差是**价格战**——二线电池厂产能过剩，特斯拉、比亚迪都在压价。电池单价2024Q1环比降5%，毛利率承压。财报还没裁到，但快了。（评分 5/10）
- **V（估值）**：PE 23.4x，PB 5.66x，市值1.7万亿。不高，但也不便宜。如果增速降到15%，这个估值没有安全边际。（评分 6/10）
- **FEV 总分**：F(7/10) + E(5/10) + V(6/10) = **18/30**

### 2. 德赛西威 (002920)

- **F（基本面）**：智能座舱国内第一，智能驾驶域控制器市占率30%。2023年营收增速50%，净利率逆势提升到11%。关键是**芯片适配能力**——高通、英伟达、地平线三家平台都跑通了。只要智能汽车L3普及加速，它的份额就是硬指标。（评分 9/10）
- **E（预期差）**：市场把它当汽车电子周期股，PE 24倍看着合理。但忽略了一点——**它是A股唯一能卡“智能驾驶芯片+软件集成”的标的**。海外对手是Mobileye、高通，但Mobileye在华份额因地缘政治风险在掉。德赛正在吃这个缺口。（评分 8/10）
- **V（估值）**：PE 23.97x，PB 3.69x，市值559亿。对标美国德赛（Aptiv）PE 25x，德赛增速更快。如果按PEG=1.5算，合理PE 35x，还有45%空间。（评分 8/10）
- **FEV 总分**：F(9/10) + E(8/10) + V(8/10) = **25/30**

### 3. 中航光电 (002179)

- **F（基本面）**：军工+新能源汽车连接器龙头，高可靠连接器市占率40%。2023年营收增速25%，净利率18%，ROE 15%。**产能利用率95%以上**，扩产周期18-24个月。下游新能源汽车增速30-50%是供需缺口持续扩大的来源。（评分 8/10）
- **E（预期差）**：市场把它当“军工标的”估值，PE 47x看起来贵。但它的新能源汽车连接器业务增速60%，占到营收40%了。**军工给10x，新能源给30x，加权合理PE 30x以上**。市场没意识到它正在变成“汽车连接器龙头”。（评分 9/10）
- **V（估值）**：PE 40.96x，PB 3.34x，市值779亿。绝对值高，但2025年预期PE 25x（机构一致），如果新能源车连接器市占率再提升，增速超预期→PE会压缩。（评分 7/10）
- **FEV 总分**：F(8/10) + E(9/10) + V(7/10) = **24/30**
