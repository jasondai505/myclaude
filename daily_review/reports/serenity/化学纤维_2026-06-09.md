⬇️（向上游反推）

**第5层：复合材料部件制造**
- 预浸料 → 铺层 → 热压罐/模压成型
- 关键：**工艺know-how**，不是随便谁都能做
- 全球格局：东丽（日）、赫氏（美）、索尔维（比）主导，中国有光威复材、中复神鹰在追赶

**第4层：预浸料 / 树脂基体**
- 碳纤维/芳纶 **+** 环氧树脂/双马树脂/聚酰亚胺
- 树脂配方是黑箱，尤其是耐高温、耐湿热的老化性能
- 全球格局：东丽、赫氏、索尔维自供为主；中国有光威（自供预浸料）、中复神鹰（也开始向下游延伸）

**第3层：高性能纤维原丝**
- **这是卡脖子核心层。** 碳纤维前驱体（PAN基）+ 纺丝 + 预氧化 + 碳化
- 全球格局：东丽（日）、帝人（日）、三菱化学（日）——**日本三家+美国Hexcel基本垄断碳纤维高端牌号**
- 中国：中复神鹰（干喷湿纺技术突破，T700/T800可量产）、光威复材（军品为主）、恒神股份
- 问题：**T1000及以上、M系列高模量碳纤维**，中国量产仍不稳定，设备禁运（氧化炉、碳化炉进口受限）

**第2层：碳纤维前驱体（PAN原丝）**
- 不是随便能做。微孔、杂质控制、牵伸工艺、放热控制……全是累积出来的经验
- 全球格局：东丽自供，中国大部分依赖自产（但质量稳定性差）
- 这一层卡的不是技术，是**良率和一致性**

**第1层：原材料（丙烯腈）**
- 从丙烯生产，国内产能充足
- 不卡脖子

### 供需缺口量化

**碳纤维（全球视角）：**
- 需求端：2024年全球碳纤维需求约13.5万吨，年复合增长15-18%
- 供给端：东丽+帝人+Hexcel + 中国合计约12万吨名义产能，但**高模量（M系列）只有东丽能批量产，年产量不到3000吨**
- 缺口：**中国每年进口高性能碳纤维约3-4万吨**，其中T800级以上自给率不到30%
- 时间窗口：中国新增千吨级碳化线要2-3年，**供需缺口至少持续到2026-2027年**

**对位芳纶：**
- 杜邦垄断全球60%+，中国进口替代率不到20%
- 缺口：防弹衣、光缆加强芯、橡胶增强需求爆发，国内供需失衡

## 第一层：全球供应链卡脖子分析

### 产业链全貌
**化学纤维**—这条链从上游石化原料开始，中间经过聚合、纺丝、后处理，终端落在**军工复合材料、航空航天、高性能工业制品、新能源装备**四个领域。

把链条拆开看：

```
石化原料（PX/己内酰胺/丙烯腈）  
→ 聚合物切片（聚酰胺PA / 聚丙烯腈PAN / 聚对苯二甲酰对苯二胺PPTA）  
→ 原丝/纤维 （碳纤维原丝 / 高强高模芳纶 / 超高分子量聚乙烯UHMWPE）  
→ 预浸料/织物 （树脂浸润 / 编织 / 表面处理）  
→ 复合材料部件 （机翼/螺旋桨/压力容器/风电叶片）  
→ 终端产品 （C919 / 火箭壳体 / 氢能储罐 / 防弹装甲）
```

**关键卡脖子层**：**原丝/纤维制造** 和 **预浸料配方**。  
全球范围内——东丽（Toray）、帝人（Teijin）、三菱化学掌握 **T1000级以上碳纤维**、**M60J级高模量碳纤维**；帝人掌握 **对位芳纶**；DSM（荷兰）掌握 **超高分子量聚乙烯** 的冻胶纺丝工艺。  
中国目前 **T800级以上碳纤维** 能做但量产稳定性仍落后东丽1-2代，**预浸料配方中的树脂体系** 长期依赖国外牌号。

### 供需缺口量化
- **需求端**：中国碳纤维需求近3年CAGR约28%（2020-2023，来源：赛奥碳纤维），2025年预计全球需求量超15万吨。
- **供给端**：全球有效产能约12万吨（2023年），中国本土产能约7万吨（含中低端），但**T800级以上高强中模级产能不足1万吨**。扩产时间表：投资一条千吨级T800线需3-4年、设备采购周期18-24个月（受制于日德设备出口管制）。
- **缺口**：高强中模级碳纤维缺口估算在 **1.5~2万吨/年**（2024-2025年），且缺口将持续到2027年新线达产。

### Serenity 的判断（第一层）
**这条链上最不可替代的节点，不是"碳纤维"这个大词，而是"高强中模级原丝（T800/IMS级）"的稳定量产能力。** 谁能在品质一致性上接近东丽T800S（拉伸强度≥5.9GPa、模量≥294GPa），谁就锁死了中国航空/航天两大赛道的材料供应。把碳纤维做成绳子那帮人一文不值，真正卡脖子的是那个能让拉伸强度和模量同时达到东丽标准的稳定工艺。

## 第三层：FE 基本面验证

### 1. 凯赛生物 (688065)

- **F（基本面）**：  
  全球长链二元酸龙头，市占率>80%。生物基聚酰胺（尼龙56/510）已完成万吨级量产，在纺丝、工程塑料领域开启替代石化尼龙6/66。公司目前两大核心壁垒：生物法长链二元酸的菌种团队+万吨级发酵工艺（扩产周期2-3年）。  
  **问题**：2022-2023年营收连续两年下滑（从31.7亿→21.1亿），主因是尼龙66替代低于预期+下游纺丝库存周期。2024年Q1收入恢复至5.3亿（同比+28%），但净利润率从28%降至13%——高价生物法在跟低价的石化版本打价格战。  
  评级：**F = 7/10**（产品壁垒极高，但短期盈利节奏被打乱）

- **E（预期差）**：
