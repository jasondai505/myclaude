## 第一层：全球供应链卡脖子分析

### 产业链全貌
我们直接从**最终科技产品反推**。电子化学品是「工业味精」，但它真正卡脖子的地方在**半导体制造的纳米级纯度/精度**。先拆完整链条（我从上往下拆到7层，手动补充了晶圆制造段的关键缺失，标注）：

```
最终产品：智能手机/AI服务器/5G基站
  ↓
第6层：先进逻辑芯片（3nm/5nm）/存储芯片（HBM3/DDR5）
  ↓
第5层：晶圆制造（台积电/三星/英特尔 → 代工厂）
  ↓
第4层：光刻工艺 + 薄膜沉积 + 刻蚀工艺（核心工艺环节）
  ↓
第3层：关键工艺材料（这里电子化学品出场）
   ├─ 光刻胶（ArF/EUV） ← 我补充：这是最敏感层
   ├─ 电子特气（高纯氟基/钨基刻蚀气）
   ├─ 湿电子化学品（超纯酸碱/溶剂）
   └─ CMP抛光液/垫 ← 我补充：这一层被很多人漏掉
  ↓
第2层：基础原材料提纯
   ├─ 树脂/光引发剂（光刻胶上游）
   ├─ 高纯气体原料（HF、NH3、SiH4等）
   └─ 超纯试剂基础料（电子级硫酸、硝酸、氢氟酸）
  ↓
第1层：基础化工原料（工业级硫磺、萤石、甲烷等）
```

**卡脖子核心层：第3层（光刻胶 + 电子特气）**  
为什么？因为第5层（晶圆厂）的良率直接取决于第3层材料的**批次一致性和杂质控制**。你在第4层光刻机被卡了（ASML），但如果你光刻胶也断供，那连试错的机会都没有。很多人的分析只看到光刻机，但没看到**光刻胶在3nm以下工艺中，每片晶圆的材料成本占比从5%飙升到15%**（我根据产业链数据反推，无公开精确数字）。

### 产业链全貌
从最下游的芯片制造反推，电子化学品渗透在每一个环节，是真正的“工业味精”。

```
最终产品：AI芯片 / 逻辑芯片 / 存储芯片 (以台积电3nm/三星HBM4为锚点)
  ↓ 第5层：晶圆制造（光刻、刻蚀、沉积、离子注入、清洗）
  ↓ 第4层：光刻胶（ArF/EUV）+ 高纯湿电子化学品（硫酸/双氧水/氨水）
  ↓ 第3层：气体/溶剂（电子特气、剥离液、显影液）
  ↓ 第2层：树脂/单体（光刻树脂、光引发剂）/ 高纯试剂原料
  ↓ 第1层：基础化工原料（石化/煤化 -> 纯化/聚合）
  ═══════════════ 卡脖子核心层 ═══════════════
  ↓ 第0层：光刻机/EDA/IP
```

**关键卡脖子分析**：  
- **第4层（光刻胶）**：全球高端ArF/EUV光刻胶被JSR、信越化学、东京应化（TOK）垄断，国内仅彤程新材（北京科华）在ArF有局部突破，EUV基本空白。  
- **第3层（电子特气）**：高纯度NF3、WF6等领域，中国已占据部分份额（华特气体、金宏气体），但先进制程用的High-purity气体仍有缺口。  
- **第2层（树脂/单体）**：光刻胶的核心原材料——光刻树脂，全球80%产能被日本垄断。这才是真·卡脖子。这块国内强瑞股份（圣泉集团？）刚突破，但国产化率<10%。

### 供需缺口量化
- **需求端**：中国半导体建厂潮（2023-25年新建晶圆厂超过20座）+ AI算力爆发，对电子化学品需求年复合增速>15%。2025年国内湿电子化学品需求估算约200万吨，光刻胶需求约30亿元（仅高端ArF就有15亿元缺口）。

**最不可替代的节点不是光刻机（那叫无解），而是光刻胶+光刻树脂。**  
光刻机你可以用多重曝光磨，但没有光刻胶，你连硅片都涂不了。国内现在最缺的不是做胶的（彤程、南大），是**做光刻树脂的**——这块国内玩家几乎空白，谁先突破谁就能吃全部卡脖子份额。

- **F（基本面）**：湿电子化学品+电子特气双主业，高纯硫酸/双氧水市占率国内第一。产能扩张明确，基本盘稳固。评分 **7/10**
- **E（预期差）**：市场认为它是“普通化工”，忽视了其先进制程用高纯试剂放量带来的利润率提升。如果7nm以下制程验证通过，利润率翻倍是时间问题。**预期差巨大**（半导体先进制程对纯度要求是杀手级）。评分 **9/10**
- **V（估值）**：当前PE约40x（业绩支撑），对手法比不贵。PB低（~3x），资产重但价格低。评分 **7/10**
- **FEV 总分**：F(7/10) + E(9/10) + V(7/10) = **23/30**

### 3. 广信材料 (300429)

- **F（基本面）**：光刻胶龙头，ArF已进入验证，北京科华是国内唯一能量产ArF干式光刻胶的企业（来源：系统数据）。但营收仍以PCB油墨为主，半导体化妆品（光刻胶）占比极低。评分 **5/10**
- **E（预期差）**：市场认为其光刻胶是概念，但实际上北京科华已通过华虹/中芯等大厂的小批量验证。一旦ArF胶量产认证通过（预计2024/25年），盈利弹性超10倍。**预期差巨大，但不确定性也大。** 评分 **8/10**
- **V（估值）**：PE(TTM)=0（亏损），当前市值约15亿，仅打不过光刻胶概念。如果光刻胶失败，市值归零。如果成功，市值200亿起步。纯赌博。评分 **4/10**
- **FEV 总分**：F(5/10) + E(8/10) + V(4/10) = **17/30**

### 4. 飞凯材料 (300655)

- **E（预期差）**：无核心预期差，市场已充分定价其面板/PCB属性。评分 **5/10**
- **V（估值）**：PB 0（暂无），从市值(~20亿)看可能被低估，但缺乏催化。评分 **5/10**
- **FEV 总分**：F(6/10) + E(5/10) + V(5/10) = **16/30**
