- **V（估值）**：PE 23.4x，PB 5.66x，市值1.7万亿。历史PE中枢40x，当前是显著折价。DCF模型（假设WACC 9%，永续增长3%）给出合理估值430-450元，上行空间约10-13%。不算低，但也不贵。**评分：7/10**
- **FEV 总分**：F(9/10) + E(7/10) + V(7/10) = **23/30**

### 2. 阳光电源 (300274)

- **F（基本面）**：全球逆变器出货量第一（市占率30%+），储能系统集成全球前五。海外收入占比60%+（欧洲+中东+亚太），毛利率稳定在30%+。**评分：8/10**
- **E（预期差）**：市场担心逆变器竞争加剧和价格战（华为+锦浪+固德威都在抢）。但市场低估了储能PCS的渗透率——电网侧储能PCS需求爆发，且阳光电源已经拿到中东大单（7.8GWh）。**评分：6/10**
- **V（估值）**：PE 26.5x，PB 6.7x，市值2420亿。逆变器板块历史PE中枢30-35x，当前估值合理偏低。但储能系统集成这块估值还没充分定价。**评分：7/10**
- **FEV 总分**：F(8/10) + E(6/10) + V(7/10) = **21/30**

### 3. 国电南瑞 (600406)

- **F（基本面）**：电网自动化+调度系统+继电保护三个领域市占率均>60%，国家电网指定供应商。每年营收稳定增长8-12%，毛利率27%+，经营性现金流>12%营收。**评分：9/10**
- **E（预期差）**：市场认为它只是「电网公用股」——但特高压+柔性直流+配电网改造三条线同时启动，国电南瑞在每条线上都是最大受益方。2025年电网投资5000亿+，国电南瑞至少拿走1000亿的订单。**评分：6/10**
- **V（估值）**：PE 22.2x，PB 3.5x，市值1845亿。作为电网垄断型科技公司，PE 22x合理偏低，历史PE在25-35x之间波动。如果按DCF（WACC 9%，永续4%）算得合理估值28-30元，当前还有25%+空间。**评分：8/10**
- **FEV 总分**：F(9/10) + E(6/10) + V(8/10) = **23/30**

### 4. 保变电气 (600550)

- **F（基本面）**：特高压变压器+换流变龙头之一（与西电、特变电工三分天下）。2024年营收约130亿，毛利率22%。问题：原材料（取向硅钢+铜+绝缘材料）+核心器件（IGBT/高压开关）大量外购，利润极薄，净利润率只有2-3%。**评分：5/10**
- **E（预期差）**：市场期待特高压建设加速→变压器订单增加。但特高压变压器并非卡脖子环节——国内能做的人不少（至少4家）。真正卡脖子的是被海外控制的IGBT和SiC，保变电气空有规模，没有核心技术壁垒。**评分：3/10**
- **V（估值）**：PE 98.9x，PB 25.3x，市值225亿。PE接近100倍，PB 25倍——绝对高估值，严重泡沫化。原因：游资炒特高压概念，题材驱动而非基本面驱动。**评分：1/10**
- **FEV 总分**：F(5/10) + E(3/10) + V(1/10) = **9/30**

### 5. 特变电工 (600089)

- **F（基本面）**：多晶硅（新特能源）+变压器+新能源EPC三大业务。2024年营收约1100亿，净利润120亿+。各项业务市场地位稳固（多晶硅国内前五、变压器国内前三）。**评分：7/10**
- **E（预期差）**：市场把它当成「多晶硅+变压器」来看——但忽略两块：①新能源电站开发转让业务（可重复产生高ROE）；②海外EPC订单（中东+中亚）。这两块合计贡献利润占比已达30%+，且增速>40%。**评分：7/10**
- **V（估值）**：PE 19.3x，PB 1.7x，市值1188亿。1.7倍PB处于历史低位，PE 19x处于行业合理区间。但多晶硅周期性波动（硅料价格从300元跌到40元/kg）影响了整体估值。拆分来看：变压器+EPC这块配10-15x PE，多晶硅周期性5-8x PE，综合合理估值仍在25元以上。**评分：8/10**
- **FEV 总分**：F(7/10) + E(7/10) + V(8/10) = **22/30**
