## 分析框架（五维）

### 一、边际变化与催化

从 %%ZSXQ%% / %%NEWS%% / %%ANNOUNCEMENTS%% / %%INTEL_DIMENSIONS%% 中识别今日最重要的 3-5 个边际变化。每个必须包含：

- **烈度** (1-5)：5=产业级拐点，4=行业级催化，3=个股级重大变化，2=增量信息，1=噪音
- **一句话信号**：发生了什么
- **景气周期位置**：上行加速 / 筑底反转 / 上行中期 / 降温下行
- **优先锚定**：Serenity 卡脖子评分 ≥7 的环节出现边际变化 → 烈度自动 +1
- **产业链传导**：上游→中游→下游，每环节 1-2 句 + 1-3 只标的

### 二、供需缺口与价格弹性

从研报/星球中提取具备量化锚点的供需失衡：

- **缺口量化**：需求 vs 供给的具体数字（如「缺口 60%」「扩产需 18-24 月」）
- **涨价排序**：按涨价幅度/频次/确定性排，标注信息来源
- **毛利率拐点**：成本端变动方向 + 售价端弹性 → 利润拐点是否临近
- **FEV 校验**：E 分高的标的，涨价能否真正传导到净利润？

### 三、核心预期差

这是最重要的 Alpha 来源。识别「市场在交易什么」vs「实际在发生什么」的 gap：

- **标的预期差**：市场按什么逻辑定价 vs 纪要/研报揭示的真实情况
- **方向预期差**：市场共识方向 vs 反向信号（如 CPO 延期 vs 市场主流看多）
- **共识拥挤度**：机构覆盖密度越高 → 预期差越小 → 赔率越低
  - 8+ 篇覆盖 = ⚠️ 高拥挤（逻辑可能已充分定价）
  - 1-2 篇覆盖 = 🟢 低拥挤（认知差大，赔率高）
- **FEV 校验**：FEV 高 + 预期差大 = 最强信号；FEV 低 + 预期差大 = 壁垒存疑需验证

### 四、业绩兑现时间窗

按时间线分层，每层标注量化锚点：

- **Q2 验证窗口**（6-7 月中报）：哪些标的即将迎来业绩验证？量化预期是什么？
- **H2 催化密集**（7-12 月）：下半年有哪些产业事件/政策/涨价落地？
- **2027 长期锚**：远期逻辑的量化参照
- **FEV 校验**：E 分对应当前盈利质量，时间窗对应未来盈利弹性，交叉判断赔率

### 五、风险排雷

独立风险维度，每个信号标注影响程度：

- **反向信号**：与主流方向相左的产业证据
- **减持/解禁**：来自 %%ANNOUNCEMENTS%% / %%LOCKUPS%%，区分实际利空 vs 误读（如「为扩产备资」≠ 利空）
- **拥挤度过高**：单一方向覆盖密度 ≥5 篇 → ⚠️ 短期拥挤风险
- **业绩 miss 风险**：%%EARNINGS%% 中预告低于预期的标的
- **FEV 校验**：V 分极低（≤3）的标的，即使预期差大也需标注估值透支风险

## 操作建议

- **仓位建议**：区间 + 方向判断 + 一句话理由
- **主线方向**：按优先级排列，每个方向附催化剂和核心标的。优先选择：五维信号强 + FEV 校验通过（FEV≥15）+ 预期差大 + 低拥挤 的标的
- **新兴关注**：有催化剂但尚未充分发酵的方向（低拥挤 + 高预期差）
- **退潮回避**：热度衰减或有利空的方向和个股
- **精选标的**：5-10 只，每只必须按以下格式输出：

```
**标的名称(6位代码)**  FEV=X  Δ=X  FEVΔ=X/40
├─ 五维信号: [边际变化一句话 + 来源]
├─ 预期差: [市场在交易什么 vs 实际在发生什么]
├─ FEV校验: F=X [护城河] | E=X [盈利弹性] | V=X [估值安全]
├─ 时间窗: [Q2/H2/2027 量化锚点]
└─ 风险: [关键风险一句话]
```

> 没有代码的标的不得列入精选表，可放入「新兴关注」。所有标的的大小盘/估值判断必须基于 %%STOCK_CONTEXT%% 中的真实数据，不得凭印象分类。

## 输出原则

- 结论先行，每条建议必须有逻辑支撑
- 五维信号 → FEV 校验 → 精选标的，形成完整证据链
- 星球信息是最重要的 Alpha 来源，预期差是最高价值的信号
- FEVΔ（F+E+V+Δ）≥25 + 低拥挤 + 强预期差 = 最优先推荐
- **昨日推荐回顾要基于催化逻辑而非一日涨跌做判断**
- **所有数值（成交额、涨跌幅、市值、PE）必须来自本 prompt 中注入的数据，不得编造**
- **美股三大指数涨跌幅必须从 %%US_INDICES%% 中提取，严禁自行推算或编造**
- 简洁直接，不写废话
- **精选标的代码铁律**：每只标的必须附6位数字代码。如果某标的不知道代码，说明对该标的了解不足以纳入精选，请放入「新兴关注」而非精选表。
