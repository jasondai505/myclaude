## ⚡ 第二层：今日信号 [五维 — 今天打哪]

本节覆盖五维框架的全部五个维度。**每个信号标注 `[五维]` 来源，引用具体数据出处**（星球#编号/研报来源/公告日期）。

### ① 边际变化与催化

从 %%ZSXQ%% / %%NEWS%% / %%ANNOUNCEMENTS%% / %%INTEL_DIMENSIONS%% 中识别 3-5 个最重要的边际变化：

- **烈度** (1-5)：5=产业级拐点，4=行业级催化，3=个股级重大变化
- **景气周期位置**：上行加速 / 筑底反转 / 上行中期 / 降温下行
- **锚定规则**：Serenity 卡脖子 ≥7 的环节出现边际变化 → 烈度 +1，标注 `[Serenity锚定]`
- **产业链传导**：上游→中游→下游，每环节 1-2 句 + 1-3 只标的

### ② 供需缺口与价格弹性

- **缺口量化**：需求 vs 供给的具体数字（标注出处）
- **涨价排序**：按幅度/频次/确定性排
- **毛利率拐点**：成本端 + 售价端 → 利润拐点

### ③ 核心预期差

最重要的 Alpha 来源。每项标注市场共识拥挤度：

- 8+ 篇覆盖 = ⚠️ 高拥挤（逻辑可能已充分定价）
- 1-2 篇覆盖 = 🟢 低拥挤（认知差大，高赔率）

### ④ 业绩兑现时间窗

- **Q2 验证**（6-7 月中报）+ **H2 催化**（7-12 月）+ **2027 长期锚**

### ⑤ 风险排雷

- 反向信号 / 减持解禁 / 拥挤过高 / 业绩 miss

## 输出原则

- **三层递进**：Serenity 定战场 → 五维定方向 → FEV+Δ 定标的，缺一不可
- **来源标注**：每个数据点标注 `[Serenity]` / `[五维]` / `[FEV]`，可追溯可审计
- 星球信息是最重要的 Alpha 来源，预期差是最高价值的信号
- FEVΔ（F+E+V+Δ）≥25 + 低拥挤 + 强预期差 = 最优先推荐
- **所有数值（成交额、涨跌幅、市值、PE）必须来自注入数据，不得编造**
- **美股三大指数涨跌幅必须从 %%US_INDICES%% 中提取**
- **精选标的代码铁律**：每只标的必须附6位代码，无代码不放精选表
